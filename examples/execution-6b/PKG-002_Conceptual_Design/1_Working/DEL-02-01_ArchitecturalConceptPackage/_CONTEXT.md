# Deliverable Context

**Deliverable ID:** DEL-02-01
**Name:** Architectural Concept Package
**Package:** PKG-002 -- Conceptual Design
**Type:** Design / Drawings

## Description

Architectural concept drawings per Section 7.1.1: building floor plan(s) showing room layout, adjacencies, and program compliance with Functional Program and room sizing ranges (Addendum 03); building elevations (all sides); key building sections; massing/3D views (optional renderings). Dimensioned drawings showing building size. Incorporates all Addendum 03 architectural requirements: 16 ft overhead doors, 40 bunker gear lockers with ventilated storage room, room sizing per addendum ranges, second-storey option for shared spaces. Excludes pickled sand storage building.

- (Optional) 3D renderings/massing diagrams
