# Specification: Detailed Project Schedule (DEL-09-01)

## Scope

### What This Deliverable Covers

The Detailed Project Schedule is a Gantt chart (PDF) and supporting narratives documenting the project timeline from design kickoff through Construction Substantial Performance, Close-Out, and the 12-month warranty review period. It demonstrates a feasible, cost-optimized timeline leveraging the Owner's stated "not schedule-driven" philosophy.

Covers:
- Gantt chart showing all key work packages with durations, milestone dates, and critical path
- Five design submission milestones per RFP Section 7.1.8
- Construction phases for both the Main Building (Public Services Building) and Cold Storage building (60' x 100')
- Commissioning, training, deficiency resolution, occupancy permit, and 12-month warranty review
- Critical path narrative explaining the critical path logic
- Schedule assumptions list stating all material assumptions

### What This Deliverable Excludes

- Pickled sand storage building (removed per Addendum 03 ยง1.2; Decomposition C-10)
- Schedule control narrative during construction (covered by DEL-07-01, Construction Methodology)
- Post-award monthly schedule updates (post-award obligation per RFP 7.2; not a proposal artifact)
- Budget or cost schedule (pricing covered in PKG-001 deliverables)

**Scope sources:** RFP Section 7.1.9 (pp. 18-19); Decomposition v2.3 ยง9 (DEL-09-01); SOW-0025.
