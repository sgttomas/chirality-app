# Procedure — DEL-001-02 Architectural Floor Plans

---

## Purpose

This procedure describes the steps for producing the Architectural Floor Plans drawing set (DEL-001-02) for the New North Shop addition at the West Dried Meat Lake Regional Landfill Maintenance Shop project. It covers the sequence from initial inputs through to IFC (Issued for Construction) drawing issuance and confirmation of deliverable completeness.

The floor plan set is the spatial anchor for all subsequent discipline drawings. Producing it correctly and coordinating it before IFC issuance is a critical-path activity for the project.

## Steps

### Step 1 — Establish project data and code analysis

1.1 Confirm project location, legal description (SW 14–44–21–W4, Camrose County, Alberta), and applicable Authority Having Jurisdiction (AHJ) for building permit purposes.
> Source: RFP §1.0, §3.3.2.

1.2 Identify the applicable edition of the Alberta Building Code with the AHJ. Determine building occupancy classification (ASSUMPTION: industrial/Group F occupancy is likely, with possible Group B or Group A sub-occupancies for office and washroom areas — confirm with Architect of Record).

1.3 Determine code-minimum requirements for:
   - Means of egress from all spaces including the mezzanine
   - Aisle and circulation clearances in industrial bays
   - Accessible design (where triggered by ABC)
   - Fire separations between occupancy groups (shop vs. office vs. storage)
   - Plumbing fixture count minimums for the anticipated occupant load

1.4 Record the code analysis results as a reference document to support the floor plan layout decisions. This document supports future building permit review.

---

### Step 2 — Establish column grid and structural interface

2.1 Engage the Structural Engineer (PKG-002 team) to confirm the column grid for the 130' × 100' building footprint based on the 35' ceiling height, 5-tonne overhead crane runway requirements, and anticipated foundation type.
> Source: RFP §3.4; R-04; Decomp SOW-0012, SOW-0025, SOW-0067.

2.2 Confirm bay widths: two repair bays at 24' wide, one wash bay at 24' wide, with remaining column grid bays accommodating the parts room, utility room, office, and washroom zones.
> Source: R-04.

2.3 Confirm crane runway beam locations and column positions. Record these on the architectural floor plan as an overlay reference layer.

2.4 Confirm service pit location and dimensions with the Structural Engineer (DEL-002-06). The pit must allow under-equipment service access and be coordinated with overhead crane hook coverage.

---

### Step 3 — Draft the ground floor plan

3.1 Set up drawing files with the confirmed column grid and building footprint (130' × 100').

3.2 Lay out all programmatic spaces per Appendix B conceptual arrangement (adjusted as required by code analysis and structural grid):
   - Two drive-through repair bays (24' wide each) with overhead doors at both ends
   - Wash bay (24' wide, enclosed, single overhead door) at one end of the building
   - Parts storage room (~400 sq ft, with 6' wide roll-up door)
   - Utility room (dimensions TBD — confirm with Mechanical/Plumbing engineers)
   - Office space (dimensions TBD — Architect to size per program)
   - Washroom
   - Industrial wash station location
   - Service pit/trench location (below grade — show as noted plan element)
   - 50,000 L cistern location (confirm dimensions with Plumbing Engineer)
   - Stairs to mezzanine

3.3 Show all entrance/exit doors for the addition (separate entrance and exit doors required per RFP §3.1).

3.4 Show welding station location in main shop area.

3.5 Show work bench locations (from Appendix B as reference).

3.6 Show steel plate locations in concrete floor. Mark as "Steel Plate — Embedded, See Structural" with an annotation reference to DEL-002-08 (Steel Plate Embedment Plan).
> Source: R-04; Decomp SOW-0024.

3.7 Show sump drain locations in repair bays and floor drain locations as applicable. Annotate with reference to plumbing drawings.
> Source: Decomp SOW-0045, SOW-0046.

3.8 Show the wash bay floor drain and connection to exterior mud sump. Show exterior mud sump location (outside building envelope, adjacent to wash bay, accessible by excavator).
> Source: R-04; Decomp SOW-0027b.

3.9 Show north arrow, scale bar, drawing title block.

3.10 Show relationship to Old North Shop (105' × 40') at west side, with interface line clearly marked.

---

### Step 4 — Draft the mezzanine floor plan

4.1 Prepare a separate mezzanine level plan showing:
   - Mezzanine extent (over parts room, utility room, and wash bay)
   - Mezzanine floor area (TBD by Architect based on structural framing and code egress requirements)
   - Stair landing and access from ground floor
   - Open-to-below edge(s) with guardrail notation
   - Loading note referencing heavy item storage capacity (oil totes, pumping equipment) — coordinate value with Structural Engineer (DEL-002-05)
   - Underside clearance note for wash bay (see Guidance T-1)

4.2 Confirm mezzanine egress with code analysis (Step 1.3).

---

### Step 5 — Show site context on floor plan key sheet

5.1 Prepare a site/key plan sheet or include site context on the first floor plan sheet showing:
   - Building orientation relative to site
   - Relationship to Old North Shop
   - Cement pad (east/south of addition)
   - Gravel pad (north/west)
   - Exterior mud sump location
   - Access roads and overhead doors (approach directions)
> Source: R-04.

---

### Step 6 — Interdisciplinary coordination review

6.1 Issue the draft floor plan (ground floor + mezzanine) to all discipline engineers for coordination review:
   - Structural: confirm column grid, crane runway positions, service pit, mezzanine framing, steel plate zones, stair structure
   - Mechanical: confirm MUA unit space, exhaust routing, ceiling fan locations (6 fans in main shop area per Decomp SOW-0040), repair bay exhaust hose and fan locations
   - Electrical: confirm crane power routing, overhead door power locations, panel/switchgear space, lighting zones (for coordination with DEL-004-04)
   - Plumbing: confirm cistern placement, drain locations, emergency shower location (DEL-006), wash station location

6.2 Resolve all coordination comments. Document decisions that deviate from the Appendix B conceptual arrangement and confirm with the Owner where program changes are involved.

6.3 Confirm interface with Old North Shop renovation (DEL-001-09/10 team).

---

### Step 7 — Code compliance and permit review preparation

7.1 Perform final code compliance check against the Alberta Building Code:
   - Egress path lengths and widths (all occupied spaces)
   - Mezzanine egress (Step 4.2)
   - Required fire-rated separations (if mixed occupancy)
   - Accessible design compliance (where required)
   - Washroom fixture count

7.2 Prepare any required code compliance documentation or schedules for inclusion in the building permit application package (coordinated with DEL-009-02 — Building Permit Application).

---

### Step 8 — IFC drawing production and stamp

8.0 **GATE CHECK:** Confirm that County approval of DEL-001-01 (Preliminary Architectural Design) has been received and is documented in the project record before proceeding with IFC production. If approval is not on file, do not proceed — escalate to the project manager. (Lensing item A-004)

8.1 Incorporate all coordination comments and code compliance revisions into the final drawing set.

8.2 Complete the drawing title block for each sheet:
   - Project name: 2026 Design Build — West Dried Meat Lake Regional Landfill Maintenance Shop Addition
   - Deliverable: Architectural Floor Plans
   - Drawn by / checked by / issued by (Architect)
   - Issue date
   - Revision number
   - Scale
   - Sheet number / total sheets

8.3 Obtain review and sign-off from the Architect of Record.

8.4 Obtain P.Eng. signature and stamp on all IFC sheets (as required by RFP §3.3.2). Confirm with the Owner and AHJ whether an architectural stamp is also required and obtain same.
> Source: RFP §3.3.2; Guidance Conflict CONF-003.

8.5 Issue drawing set as IFC. Record issue date and distribution in the project document register.

---

### Step 9 — Transmit to Owner and project team

9.1 Transmit IFC drawing set to:
   - Camrose County (Owner) — through County project manager
   - General Contractor / construction team
   - All discipline engineers (for construction coordination)
   - AHJ as part of building permit package (DEL-009-02)

9.2 File original stamped drawings in the project record.
