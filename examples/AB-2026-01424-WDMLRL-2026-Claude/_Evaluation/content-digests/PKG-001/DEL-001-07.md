## DEL-001-07 — Door & Window Schedule

### Identity
- **Package:** PKG-001 Architectural Design
- **Discipline:** Architect
- **Type:** Schedule
- **Responsible:** Architect

### Scope
**Description:** Door & Window Schedule covering SOW-0011 (Complete final architectural design) and SOW-0026 (Install separate entrance/exit doors for the addition) for the 2026 WDMLRL Maintenance Shop Addition. The schedule must enumerate all door and window openings in the new ~13,000 sq ft concrete facility and renovation areas.

**Acceptance Criteria:**
- All door openings enumerated with tag, location, type, nominal size (W×H), frame material, operation, fire rating, hardware group, finish, and remarks
- Repair bay and wash bay overhead doors sized minimum 24' width (height TBD pending Owner equipment specifications)
- Parts storage roll-up door minimum 6' width
- Separate entrance/exit doors identified per SOW-0026
- Fire ratings assigned per Alberta Building Code analysis
- Cross-references to architectural floor plans (DEL-001-02)
- Compliance with all applicable Alberta building codes and regulations

**SOW IDs:** SOW-0008, SOW-0009, SOW-0011, SOW-0026, SOW-0072, SOW-0073, SOW-0018

**OBJ IDs:** OBJ-001 (code-compliant facility), OBJ-003 (complete P.Eng.-stamped IFC design documentation)

### Document Kit Summary
- **Datasheet:** Catalogs 12 known door openings from conceptual sources with TBD values for heights, fire ratings, and hardware groups; identifies no windows in available sources (Architect to determine per code); flags critical input dependency (Owner equipment dimensions).
- **Specification:** Defines six core requirements (REQ-DW-001 through REQ-DW-006) including separate entrance/exit doors, overhead door heights, roll-up door width, schedule completeness, and Alberta Building Code compliance; notes gaps in equipment clearance criteria and schedule presentation standards.
- **Guidance:** Articulates five key design principles (industrial character, entrance/exit separation, structural coordination, electrical coordination, fire rating determination) and outlines hardware selection criteria for industrial use; emphasizes Owner coordination on equipment heights and code analysis for fire/egress/window requirements.
- **Procedure:** Provides eight-step workflow from code analysis through construction-phase use; includes prerequisites (preliminary design, owner confirmation, code analysis, site investigation, multi-discipline coordination) and specific coordination checkpoints with structural, mechanical, and electrical packages.
