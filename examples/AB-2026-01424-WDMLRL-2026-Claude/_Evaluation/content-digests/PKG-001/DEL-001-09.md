## DEL-001-09 — Old North Shop Demolition Plans

### Identity
- **Package:** PKG-001 Architectural Design

- **Type:** Drawing Set
- **Responsible:** Architect

### Scope
- **Description:** Drawing set deliverable documenting demolition scope for three existing building renovation areas (kitchenette ~250 sq ft, mezzanine, washroom) within the Old North Shop, enabling renovation design and supporting permit applications.
- **Acceptance Criteria:**
  - All three renovation areas (SOW-0070, SOW-0071, SOW-0072) clearly identified and annotated
  - Existing conditions documented with elements to be removed delineated using standard graphic conventions
  - Demolition extents aligned with renovation design intent (DEL-001-10)
  - IFC stamping by P.Eng. licensed in Alberta
  - Code compliance verified (Alberta building codes and Safety Codes)
  - Preliminary design approved by County before IFC issue
- **SOW IDs:** SOW-0070 (Kitchenette), SOW-0071 (Mezzanine), SOW-0072 (Washroom)
- **OBJ IDs:** OBJ-001 (Code-compliant renovations), OBJ-003 (Complete P.Eng.-stamped IFC documentation)

### Document Kit Summary
- **Datasheet:** Comprehensive identification of deliverable with existing building attributes, renovation areas, drawing set attributes, and conditions spanning 80 lines of structured metadata.
- **Specification:** 10 requirements defining coverage of all three areas, existing conditions documentation, clear scope delineation, coordination with renovation plans, IFC stamping, code compliance, preliminary design approval, kitchenette minimum area, permit-readiness, and field verification sign-off.
