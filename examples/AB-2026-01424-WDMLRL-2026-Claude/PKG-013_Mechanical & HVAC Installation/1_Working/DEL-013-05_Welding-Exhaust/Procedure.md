# Procedure — DEL-013-05: Welding Station Exhaust System

---

## Purpose

This procedure describes the steps to produce and deliver the Welding Station Exhaust System for the New North Shop at the West Dried Meat Lake Regional Landfill Maintenance Shop Addition. It covers design coordination, procurement, installation, testing, commissioning, and records submission required to satisfy SOW-0039, OBJ-001 (code-compliant installation), and OBJ-004 (operational readiness).

## Steps

### Phase 1 — Design

**Step 1.1 — Receive County Welding Equipment Specifications**
Obtain welding equipment specifications from the County (per R-01 §3.4). Record welding processes (MIG, TIG, stick, flux-core, etc.), base metals, and expected operating frequency. This information governs fume character, capture velocity requirements, and filtration specification.

**Step 1.2 — Confirm Welding Station Layout**
Confirm final location and count of welding station(s) with County and project team. Verify against Appendix B (R-04) floor plan. Record confirmed station locations as design input. See Guidance CT-002 regarding station count conflict.

**Step 1.3 — Mechanical Engineer Designs LEV System**
The Mechanical Contractor engages a P.Eng.-licensed mechanical engineer (Alberta) to design the welding exhaust LEV system. Design shall address:
- Hood type and positioning (source-capture distance, capture velocity)
- Ductwork layout (flexible connections, rigid duct routing)
- Filtration unit selection (type, efficiency, face velocity, maintenance access) — using Specification Filtration Selection Criteria
- Exhaust fan selection (CFM, static pressure, motor size)
- Exhaust stack location and height (above roofline, clear of air intakes and building openings); identify specific code clause establishing minimum height
- Controls and fan interlock decision (see Specification REQ-007 note regarding interlock evaluation)
- Airflow coordination with DEL-013-03 (Makeup Air) and confirmation of building pressure balance
- Coordination with DEL-013-04 (Heavy Equipment Exhaust) for duct routing and aggregate exhaust volume
- Fire/spark hazard evaluation per Specification REQ-013
- Confirm applicable standard editions and clause references (see Specification Standards note [A-002])
- Quantify approximate heat loss per unit of exhaust airflow at design winter temperature (see Guidance Cold Climate Stack Considerations [F-003])
- Define quantitative acceptance thresholds for commissioning (capture velocity, airflow CFM tolerance, stack height) per Specification Verification [A-003]
- Specify commissioning test method for source capture verification (smoke test, tracer gas, or anemometer) per Specification Verification [X-005]

All IFC drawings must be P.Eng.-stamped and Alberta-licensed per R-01 §3.3.2. See Specification Documentation "P.Eng. Stamp Scope" for open questions on stamp coverage beyond IFC drawings.

**Step 1.4 — Preliminary Design Review with County**
Submit preliminary design to the County project representative for review and approval, consistent with R-01 §3.3.2 ("Deliver Preliminary Design for the County project team to approve").

**Step 1.5 — Obtain Safety Code Permit**
Apply for and obtain applicable mechanical Safety Code permit from the Alberta Authority Having Jurisdiction (AHJ). Note: permit fees are to be paid by the County per R-01 §3.3.1 ("Pay for development permit, building permit, and Safety Code Permits Fees"); the Proponent is responsible for obtaining permits per R-01 §3.3.2. Confirm required AHJ inspection hold points at permit application (see Step 3.9 and note [A-004]).

**Step 1.6 — Coordinate Exhaust Outlet Location Approval**
Confirm exhaust outlet and stack location with AHJ and any applicable environmental authority (including EPEA review if required per Specification REQ-010 note [C-002]). Obtain required approvals before finalizing penetration locations with PKG-011 (Building Structure & Envelope).

### Phase 2 — Procurement

**Step 2.1 — Procure Equipment and Materials**
Procure exhaust hood(s), flexible ductwork, rigid ductwork (if applicable), filtration unit, exhaust fan, stack/outlet fitting, controls, hangers, and fasteners per IFC specification. Obtain equipment submittals/cut sheets for review.

**Step 2.2 — Submittal Review**
Submit equipment data sheets and material specifications to the design engineer and County representative for review prior to installation. Confirm filtration unit meets regulatory requirements.

**Submittal review acceptance criteria** — the reviewer should verify the following against Specification requirements [X-001]:

| Check Item | Specification Reference |
|---|---|
| Hood type and capture distance match design | REQ-001, REQ-002 |
| Flexible ductwork length and diameter per design | REQ-003 |
| Filtration unit efficiency meets Alberta OHS Code requirements | REQ-004 |
| Filtration unit allows filter access and replacement | REQ-008 |
| Fan CFM and static pressure meet design specification | REQ-005 |
| Controls include on/off and status indication | REQ-007 |
| Materials are fire-safe per fire/spark evaluation | REQ-013 |
| Manufacturer warranty meets two-year minimum | REQ-012 |

---

### Phase 3 — Installation

**Step 3.1 — Coordinate Building Envelope Penetrations**
Coordinate with PKG-011 General Contractor for ductwork penetration locations through the building envelope (exhaust stack penetration). Ensure penetrations are framed, flashed, and sealed per architectural and mechanical requirements.

**Step 3.2 — Install Exhaust Hood(s)**
Install exhaust hood(s) at the confirmed welding station location(s) per IFC drawing. Mount hood at the correct height and orientation to achieve design capture distance.

**Step 3.3 — Install Flexible Ductwork Connections**
Connect flexible ductwork from hood collar to the rigid duct or filtration unit inlet per IFC drawing. Ensure flexible duct length allows the specified workstation repositioning range. Secure with appropriate fittings to prevent air leakage. [X-002]

**Step 3.4 — Install Filtration Unit**
Mount and connect filtration unit per manufacturer's instructions and IFC drawing. Confirm access for filter inspection and replacement is unobstructed.

**Step 3.5 — Install Rigid Ductwork (if applicable)**
Install rigid duct sections from filtration unit to exhaust fan and fan to exhaust outlet/stack per IFC drawing. Support ductwork at intervals per mechanical code. Seal all joints. [X-002]

**Step 3.6 — Install Exhaust Fan**
Mount exhaust fan per IFC drawing. Connect to building electrical system (coordinate with PKG-015 Electrical for motor power circuit).

**Step 3.7 — Install Exhaust Stack and Outlet**
Install exhaust stack/outlet through building envelope penetration per IFC drawing. Flash and seal penetration. Install weather cap or rain hood on exterior termination. Confirm height above roofline meets code minimum (per design engineer's identified code clause). [X-002]

**Step 3.8 — Install Controls**
Install on/off controls and any monitoring or interlock equipment per IFC drawing. Coordinate with PKG-015 Electrical for wiring. Test control wiring before energizing.

**Step 3.9 — Photograph Concealed Work**
Before closure of any concealed installations (ductwork routing through walls/ceiling, penetration flashing), photograph the installation to create a photographic record. Include images in the documentation package per Specification Documentation. [X-003]

**Step 3.10 — AHJ Inspection**
Request Safety Code inspection(s) from AHJ at the hold points confirmed during permit application (Step 1.5). The following steps are identified as potential AHJ hold points (TBD: Mechanical Contractor to confirm specific hold points with AHJ at permit application) [A-004]:

| Potential Hold Point | Phase | Rationale |
|---|---|---|
| Ductwork rough-in before concealment | Phase 3, after Steps 3.3–3.5 | Ductwork routing and connections must be inspectable before walls/ceilings are closed |
| Exhaust stack penetration | Phase 3, after Step 3.7 | Building envelope integrity and stack height verification |
| System completion before commissioning | Phase 3, after Step 3.8 | Complete installation inspection before energizing |

Provide copies of inspection reports to County representative per R-01 §3.3.2. Correct any deficiencies identified by the AHJ before proceeding.

---

### Phase 4 — Testing and Commissioning

**Step 4.1 — Pre-Commissioning Checks**
Before system start-up, verify:
- All ductwork joints sealed (no visible gaps or open connections)
- Filtration unit filter elements installed per manufacturer spec
- Exhaust fan rotates freely and motor connections correct
- Controls wiring complete and inspected
- Exhaust stack/outlet cap clear and weather-tight
- Building envelope penetration sealed
- AHJ inspection(s) passed with no outstanding deficiencies

**Step 4.2 — Initial System Start-Up**
Energize exhaust fan. Confirm fan rotates in correct direction. Listen and observe for unusual vibration or noise. Run system for a minimum initial warm-up period (TBD per manufacturer recommendation).

**Step 4.3 — Airflow Measurement**
Measure airflow (CFM) at the hood and at the fan discharge. Compare measured values to design specification. Record results. If measured airflow deviates from design CFM by more than the acceptable tolerance defined by the mechanical engineer at design stage (see Specification Verification REQ-005; TBD: specific tolerance percentage or CFM range to be established at IFC design stage), investigate and correct (ductwork leak, fan rotation, filter loading, etc.). [A-005]

**Step 4.4 — Capture Velocity Verification**
Verify hood capture velocity at the design capture distance using the test method specified by the mechanical engineer in the commissioning plan (smoke test, tracer gas test, or anemometer traverse — see Specification Verification REQ-001 [X-005]). Compare measured capture velocity to the quantitative acceptance threshold established at design stage. Record results.
ASSUMPTION: Specific capture velocity protocol and acceptance threshold TBD per mechanical engineer's commissioning plan, informed by ACGIH tables for the confirmed welding process types.

**Step 4.5 — Makeup Air Balance Confirmation**
With the welding exhaust system running, confirm the building is not in excessive negative pressure. Coordinate with DEL-013-03 (Makeup Air System) commissioning to confirm airflow balance.

**Step 4.6 — Controls and Interlock Functional Test**
Test on/off control operation. Test any interlocks (if installed per REQ-007 design decision). Confirm monitoring indicators function correctly.

**Step 4.7 — Filtration Performance Check**
If filtration efficiency testing is required by the AHJ or is specified by the mechanical engineer (TBD), conduct test and record results. Confirm filtration meets requirements before issuing commissioning sign-off.

**Step 4.8 — Commissioning Sign-Off**
The mechanical engineer (or designated commissioning agent) reviews commissioning data against quantitative pass/fail thresholds established at design stage and issues a written commissioning report confirming system performance. County representative reviews and accepts report.

---

### Phase 5 — Closeout

**Step 5.1 — Produce As-Built Drawings**
Record all as-built conditions (hood location, ductwork routing, fan location, stack location, dimensions) on as-built drawings. Submit to County.

**Step 5.2 — Compile O&M Manual**
Compile operation and maintenance manual including:
- System description and component list
- Equipment cut sheets (fan, filtration unit, hood, controls)
- Filter replacement schedule and procedure (defines ongoing maintenance frequency and responsibility post-handover; see Guidance CT-004 for one-time vs. recurring maintenance verification scope)
- Cleaning procedure for ductwork and hood
- Troubleshooting guide
- Manufacturer warranties

**Step 5.3 — Collect and Verify Warranty Certificates**
Collect warranty certificates from all equipment manufacturers (fan, filtration unit, controls). Verify each certificate provides coverage for a minimum of two (2) years from the Construction Completion Certificate date per Specification REQ-012. Include certificates in the closeout documentation package. [E-004]

**Step 5.4 — Submit Documentation Package**
Submit completed documentation package to County including:
- As-built drawings
- O&M manual
- Commissioning report
- Safety Code inspection reports
- Alberta OHS Code compliance documentation
- Environmental compliance documentation (EPEA review if applicable)
- Photographic record of concealed work [X-003]
- Warranty certificates [E-004]

**Step 5.5 — Deficiency Correction**
Correct any outstanding deficiencies noted by the AHJ or County representative. Obtain written confirmation of deficiency resolution before requesting final acceptance.

**Step 5.6 — Final Acceptance and CCC**
On completion of all work, request County final inspection. Upon County satisfaction, Construction Completion Certificate (CCC) process proceeds per R-01 §2.14. Warranty period (2 years) commences on CCC date per R-01 §2.10.2.

## Verification

| Check | Method | Acceptance Criterion |
|---|---|---|
| Hood installed at correct location | Visual inspection vs. IFC drawing | Hood position matches drawing; within design capture distance of welding point |
| Flexible ductwork connected and sealed | Visual inspection | No visible air leakage at connections; repositioning range demonstrated |
| Filtration unit accessible | Physical check | Filter elements can be accessed and removed without system disassembly |
| Airflow at design CFM | Instrumented measurement (anemometer / pitot traverse) | Measured CFM within acceptable tolerance of design CFM (TBD: tolerance to be defined by mechanical engineer at design stage) [A-005] |
| Capture velocity at hood | Test method per commissioning plan (smoke test, tracer gas, or anemometer at design capture distance) | Meets or exceeds minimum capture velocity threshold (TBD: numeric value to be established by mechanical engineer per ACGIH tables) [X-004] |
| Exhaust stack height | Tape measurement from roofline | Meets minimum code requirement (TBD: specific code clause to be identified at design stage) [E-003] |
| Controls operational | Functional test | On/off control operates correctly; status indication visible; interlock functions (if installed) |
| Building pressure balance | Manometer or equivalent | No excessive negative pressure with exhaust system running and makeup air balanced |
| AHJ inspections passed | Inspection reports | No outstanding inspection deficiencies |
| Commissioning report accepted | Document review | Mechanical engineer and County representative sign off |
| Concealed work photographed | Document review | Photographic record exists for all concealed ductwork and penetrations [X-003] |
| Warranty certificates collected | Document review | All equipment manufacturer warranties verified for two-year minimum coverage [E-004] |
