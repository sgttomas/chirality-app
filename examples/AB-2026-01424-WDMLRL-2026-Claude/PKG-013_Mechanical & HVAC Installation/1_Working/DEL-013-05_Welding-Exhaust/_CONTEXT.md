## Description
Installation of dedicated welding station exhaust system providing safe fume and particulate removal at welding workstations. System captures hot fumes and metal particulates generated during welding operations.

## Scope
- Welding station exhaust hood installation
- Flexible exhaust ductwork connections to stations
- Filtration system for welding fumes and particulates
- Exhaust fan installation
- Exhaust outlet and stack installation
- Controls and monitoring
- Performance testing and commissioning

## Key Requirements
- Safe capture of welding fumes at source
- Proper metal particulate filtration
- Code-compliant and OSHA-compliant installation
- Environmental compliance
- Proper outlet location and height
