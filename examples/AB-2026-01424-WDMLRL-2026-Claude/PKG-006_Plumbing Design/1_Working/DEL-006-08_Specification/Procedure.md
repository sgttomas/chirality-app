# Procedure — DEL-006-08 Plumbing Specification

---

## Purpose

This procedure describes the steps to produce DEL-006-08 (Plumbing Specification) — from initiation through to IFC-ready specification and PKG-014 handoff. It also references the steps to commission and verify the systems specified herein after installation by PKG-014.

**Interpretation note:** This Procedure addresses both (a) producing the specification artifact and (b) verifying/commissioning the installed plumbing systems. The specification production steps are primary; commissioning steps are included because the specification must contain them and the Plumbing Engineer is responsible for defining them.

Test parameters: (a) test pressure — TBD per applicable Alberta Safety Code (Plumbing) edition (ASSUMPTION: typically 1.5x working pressure or as specified by code; Plumbing Engineer to confirm), (b) hold duration — TBD per code (ASSUMPTION: typically minimum 2 hours for potable water systems; Plumbing Engineer to confirm), (c) pass/fail criteria — no pressure drop exceeding code allowance during hold period, no visible leaks. | Plumbing Engineer / Inspector | Pressure test record with pass/fail determination | SPEC-PLB-001; Safety Code — edition TBD — [Enrichment: C-003 — added specific test parameters] |

| 5.4 | Witness activation test of emergency shower: confirm flow rate, coverage, supply temperature, and duration meet SPEC-PLB-072 quantitative acceptance criteria and applicable Safety Code requirements. | Plumbing Engineer / Safety Codes Officer | Emergency shower test record | SPEC-PLB-070, 071, 072 — [Enrichment: F-002 — referenced quantitative criteria] |
