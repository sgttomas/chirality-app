# Procedure — DEL-006-03: Drain & Vent Plans

---

## Purpose

This procedure describes the steps required to produce the Drain & Vent Plans drawing set (DEL-006-03) for the New North Shop Addition. The procedure covers the sequence from receiving input documents through to issuing IFC (Issued for Construction) drawings and obtaining the Alberta Safety Code plumbing permit. It is intended for use by the Plumbing Engineer responsible for this deliverable.

The drawing set must be P.Eng.-stamped and produced as part of the coordinated multi-discipline IFC package before construction begins (SOW-0018; SOW-0016; RFP §3.3.2).

## Steps

### Phase 1 — Project Setup and Information Gathering

**Step 1.1 — Confirm scope and preliminary program**
- Review App B-Plumb (conceptual drawing) to document all plumbing elements shown: floor drain locations, emergency shower, wash station drain, washroom drain, septic tank location, mud sump.
- Review RFP §3.4 and SOW-0016, SOW-0027b, SOW-0043, SOW-0045, SOW-0046, SOW-0048, SOW-0050 to confirm all plumbing scope items to be addressed in the drain/vent plans.
- Confirm with Architect (DEL-001-02) the fixture list for washroom, wash station, and any other fixtures generating drainage.
- Source: App B-Plumb; RFP §3.4; Decomposition SSOW Section F.

**Step 1.2 — Confirm governing code edition**
- Contact the local Safety Codes authority (Alberta) to confirm the specific edition of the National Plumbing Code / Alberta Plumbing Code to be used for this permit jurisdiction.
- Record the confirmed code edition.
- Source: SOW-0007, SOW-0009; RFP §3.3.2.

**Step 1.3 — Site investigation (Old North Shop)**
- Conduct or arrange a site visit to the existing Old North Shop to document existing drain and vent conditions in the washroom area to be renovated (SOW-0072).
- Record existing pipe sizes, materials, invert elevations, connection point to existing septic.
- Source: SOW-0072; App B-Plumb legend (Old North Shop renovation).

---

### Phase 2 — Preliminary Design (for County Approval)

**Step 2.1 — Establish floor drain layout**
- Working from the architectural floor plan (DEL-001-02) and conceptual drawing (App B-Plumb), confirm and finalize floor drain locations for:
  - Main shop repair bays (minimum: per App B-Plumb, approximately 3 drains)
  - Wash bay (1 floor drain connecting to mud sump)
  - Emergency shower (drain at location shown on App B-Plumb)
  - Wash station (drain at location shown on App B-Plumb)
  - Washroom (floor drain / fixture drain per fixture type)
  - Sump drains in repair bays (SOW-0045)
- Coordinate drain locations with Structural Engineer for slab sleeve placement (requires early coordination before slab design is finalized).
- Source: App B-Plumb; SOW-0045, SOW-0046; REQ-006-03-010 to 013.

**Step 2.2 — Design drainage routing**
- Establish the main building drain path from all fixture/floor drain locations to the new septic holding tank connection point.
- Confirm exterior grade elevations (from Civil, DEL-005-02) to verify gravity flow is achievable from building drain to new septic holding tank.
- Establish the wash bay drain path separately, routing to the exterior mud sump.
- Identify the service pit/trench drainage provision — determine if a floor drain in the pit is required and, if so, its routing.
- Source: SOW-0027b; SOW-0043; REQ-006-03-010 to 017.

**Step 2.3 — Pipe sizing and slope calculations**
- **Pre-step verification (Lensing item X-003):** Before commencing pipe sizing, confirm that plumbing fixture unit load data has been received from DEL-006-06 (Plumbing Fixture Schedule). Record receipt of this data. If fixture unit load data is not yet available, pipe sizing cannot proceed — coordinate with DEL-006-06 to obtain this input.
- Size all drain pipes per applicable plumbing code based on fixture unit loads and flow rates.
- Calculate pipe slopes to meet minimum code requirements.
- Size vent pipes per code.
- Determine cleanout locations and spacings per code (see Specification REQ-006-03-025).
- Produce slope diagram showing pipe inverts at key points.
- Source: Applicable plumbing code (edition TBD; ASSUMPTION: NPC or Alberta equivalent); DEL-006-06 (fixture unit loads).

**Step 2.4 — Design vent system**
- Determine vent stack location(s) based on drain layout and code requirements.
- Route vent piping from all traps to vent stacks.
- Determine roof penetration locations and coordinate with architectural/structural for roof design (DEL-001-06 reflected ceiling plans, structural roof design).
- Confirm vent stack heights above roof per code.
- Source: REQ-006-03-004; ASSUMPTION: vent design governed by applicable plumbing code.

**Step 2.5 — Produce preliminary drawings**
- Produce preliminary drain & vent plan drawings (plan view minimum) showing:
  - Floor drain locations and labels
  - Drain pipe routing with pipe sizes and slopes noted
  - Vent stack locations
  - New septic holding tank connection point
  - Mud sump connection point (wash bay drain)
- Package with other preliminary plumbing deliverables (DEL-006-01) for County submission.
- Source: SOW-0010f; RFP §3.3.2.

**Step 2.6 — Submit preliminary design for County approval**
- Submit preliminary plumbing design package to Camrose County for review and approval.
- Record County approval. Address any County comments before proceeding to IFC.
- Source: SOW-0010f; RFP §3.3.2.

---

### Phase 3 — IFC Drawing Production

**Step 3.1 — Address preliminary design comments**
- Incorporate all County feedback and internal coordination comments from the preliminary design phase.

**Step 3.2 — Produce IFC drain and vent plan drawing sheets**
- Produce the complete IFC drawing set including:
  - Plan view drain layout (all floor drains, drain pipe routing, pipe sizes, slopes, inverts, cleanouts, connections to septic and mud sump)
  - Plan view vent layout (vent stack locations, vent pipe routing, vent sizes)
  - Isometric diagram(s) of drain/vent system
  - Detail sheets (floor drain assembly, cleanout detail, mud sump connection, service pit drain detail, trap details)
  - Slope diagram (building drain profile with inverts)
  - Material and equipment schedule (pipe materials, floor drain types, cleanout types)
  - General and specific construction notes
- Source: REQ-006-03-003, REQ-006-03-004; _CONTEXT.md Anticipated Artifacts.

**Step 3.3 — Cross-discipline coordination review**
- Conduct coordination check with Architectural, Structural, Mechanical, and Civil disciplines:
  - Confirm all sleeve locations match structural drawings
  - Confirm floor drain locations match architectural floor plan
  - Confirm mud sump connection grade matches civil grading plan
  - Confirm no conflicts with mechanical equipment or ductwork
- Source: REQ-006-03-030; ASSUMPTION: required coordination for multi-discipline IFC package.

**Step 3.4 — Internal quality review**
- Plumbing Engineer conducts internal review of drawings against:
  - Applicable plumbing code requirements
  - All requirements in this Specification (REQ-006-03-001 to REQ-006-03-033, including new REQ-006-03-005, REQ-006-03-025, REQ-006-03-026, REQ-006-03-027)
  - Cross-document consistency with DEL-006-02 (Water Distribution), DEL-006-04 (Cistern/Pump Details), DEL-006-05 (Septic Details)
- Record review completion.

**Step 3.5 — P.Eng. stamp and sign drawings**
- Plumbing Engineer stamps and signs all IFC Drain & Vent Plan drawing sheets.
- Confirm stamping engineer is currently licensed as P.Eng. in Alberta.
- Source: SOW-0018; REQ-006-03-001.

**Step 3.6 — Issue IFC drawing set**
- Issue Drain & Vent Plans as part of the coordinated IFC plumbing drawing set.
- Distribute to: General Contractor / Design-Builder, Plumbing Installer (PKG-014), County project representative, permit authority.
- Source: SOW-0018; RFP §3.3.2.

---

### Phase 4 — Permitting and Construction Support

**Step 4.1 — Support Safety Code permit application**
- Provide IFC Drain & Vent Plans (and supporting calculations) to the permit coordinator (PKG-009) for submission with the plumbing Safety Code permit application.
- Source: SOW-0007; REQ-006-03-022.

**Step 4.2 — Respond to permit authority questions**
- Address any technical questions from the Alberta Safety Codes permit authority regarding the drain and vent design.
- Issue drawing revisions if required by permit authority; re-stamp revised drawings.
- Source: SOW-0007; REQ-006-03-020 to 022.

**Step 4.3 — Construction support**
- Respond to Requests for Information (RFIs) from the plumbing installer (PKG-014) during construction.
- Issue field clarifications or drawing revisions as required.
- Note: Site inspections by the plumbing installer must be requested; County representative to be present at all inspections (SOW-0084).
- Source: SOW-0084; RFP §3.3.2.
