# Specification — DEL-006-05 Septic System Details

---

## Scope

### Included

This deliverable covers the complete Plumbing Engineer drawing set for the septic system associated with the New North Shop Addition at the West Dried Meat Lake Regional Landfill site. The drawing set shall document:

- Septic holding tank: supply and installation of a 2,000 US gallon septic holding tank (Source: RFP §3.4; SOW-0043)
- Removal of the existing septic tank from the site (Source: SOW-0043; Decomposition D-002)
- Septic tank location plan relative to the New North Shop building (Source: App B — Plumbing, conceptual)
- All pipe connections, cleanouts, access risers, and appurtenances required for a complete and functional septic system
- Coordination notes with related plumbing systems (floor drains, building drain and vent, washroom fixtures)
- Installation notes referencing applicable Alberta Safety Code requirements
- Site plan showing system location and required clearances

### Excluded

- Relocation of the existing septic tank to another site location — this is OUT of scope and the County's responsibility (Source: SOW-0205; Decomposition D-002)
- **Drain field, soil absorption system, distribution box, or any effluent treatment/disposal infrastructure** — this deliverable covers a holding tank only (pump-out service model). The _CONTEXT.md anticipated artifacts list references drain field and soil absorption items, but these are not applicable to the holding-tank-only scope established by RFP §3.4 and confirmed in Guidance Principle 1. If AHJ determines a treatment system is required, this would constitute a scope change requiring Owner approval. (Source: RFP §3.4; Guidance Principle 1; Lensing B-004, C-002)
- Cistern and pump system design — covered under DEL-006-04
- Mechanical HVAC design — covered under PKG-003
- Electrical design for any septic system controls or alarms — to be coordinated with PKG-004; electrical drawing deliverables under DEL-004 series
- Geotechnical investigation — covered under DEL-008-01
- Civil/site grading — covered under PKG-005
