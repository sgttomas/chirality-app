# Procedure — DEL-005-03: Drainage Plan

## Purpose

This procedure describes the steps to produce the Drainage Plan drawing set (DEL-005-03) for the West Dried Meat Lake Regional Landfill Maintenance Shop Addition. It covers the workflow from preliminary data gathering through IFC drawing production, internal verification, and handoff. It is intended for the Civil Engineer responsible for producing this deliverable within PKG-005 Civil Design.

## Steps

### Phase A — Site Familiarization and Baseline Data Collection

**Step A1 — Review project sources.**
Review the following:
- RFP §3.3.2 and §3.4 for civil design requirements, particularly drainage requirements (Source: R-01).
- Decomposition PKG-005 for deliverable scope (SOW-0015, SOW-0020, SOW-0021).
- Appendix C site maps (R-07) for site location and aerial context.
- Appendix B conceptual shop drawing (R-04) for building footprint and exterior pad locations.
- Geotechnical investigation report (DEL-008-01) for subgrade characteristics, permeability, and frost depth.
- Preliminary site survey (DEL-008-02) for existing topography, existing drainage patterns, and benchmarks.

**Step A2 — Identify existing drainage conditions.**
Document the pre-development drainage patterns at the project site boundary. Record flow directions, estimated grades, and any existing drainage features (ditches, culverts, watercourses). This baseline is required to demonstrate that post-development drainage does not alter neighbouring property conditions (REQ-003; R-01, §3.4).

**Step A3 — Confirm governing design storm and regulatory requirements.**
Determine the applicable design storm return period and rainfall intensity parameters under Alberta standards and any applicable Camrose County requirements. Document the selected design storm in the Calculation Package (DEL-005-06). This step resolves Conflict C-001 from the Guidance Conflict Table.

**Step A4 — Identify and locate the wash bay exterior mud sump.**
Confirm the planned location of the wash bay exterior mud sump with the design team. Establish the interface boundary between wash bay process drainage (SOW-0027b / PKG-018) and the clean stormwater system. This resolves Conflict C-002 from the Guidance Conflict Table.

---

### Phase B — Drainage System Design

**Step B1 — Develop drainage concept.**
Based on site topography, building layout, and driving surface arrangement, develop the conceptual drainage approach: primary flow paths, collection locations, conveyance routes (surface or piped), and disposal/dispersal method. Document the selected approach for the drainage concept discussion in DEL-005-06 Calculation Package and for inclusion in the preliminary design (DEL-005-01).

**Step B2 — Size drainage elements.**
Using the selected design storm and site parameters, calculate:
- Runoff volumes and peak flow rates (rational method or equivalent).
- Required swale or channel capacities.
- Inlet and catch basin sizing and spacing.
- Outlet structure and energy dissipation requirements.
- Detention or retention storage sizing (if applicable).
Record all calculations in DEL-005-06 (Civil Calculation Package).

**Step B3 — Coordinate with Site Grading Plan (DEL-005-02).**
Ensure that drainage flow directions, collection point locations, and surface grades on the drainage plan are consistent with the Site Grading Plan. Spot elevations, swale inverts, and grades used on both plans must agree. Resolve any discrepancies before IFC drawing production.

**Step B4 — Coordinate wash bay exterior mud sump and process drainage.**
Show the wash bay exterior mud sump location on the drainage plan with sufficient detail to support construction (SOW-0027b). Confirm with the project team that the wash bay exterior mud sump detail is clearly delineated from the clean stormwater system. Coordinate with PKG-018 construction team for constructability.

**Step B5 — Confirm neighbour drainage non-alteration.**
Verify that the designed drainage system does not export additional runoff volumes or change runoff timing at the site boundary in a way that would adversely affect neighbouring properties (REQ-003). Document this comparison in DEL-005-06.

---

### Phase C — Drawing Production

**Step C1 — Establish drawing set structure.**
Determine the number of sheets required (TBD by Civil Engineer based on complexity). Standard drawing sheet components typically include:
- Overall drainage plan (site-wide).
- Storm sewer layout and profiles (if piped system is used).
- Inlet/catch basin location plans and details.
- Outlet structure and energy dissipation details.
- Wash bay exterior mud sump location and detail.
- Drainage legend, general notes, and design storm reference.

**Step C2 — Produce drainage plan drawings.**
Draft the drawing set using CAD or equivalent engineering software (specific software TBD per firm's standard). Drawing content to reflect all designed elements, including:
- Flow arrows indicating primary drainage directions.
- Swale, ditch, or channel locations with invert elevations and grades.
- Inlet, catch basin, and manhole locations with rim and invert elevations.
- Outlet structure locations and types.
- Wash bay exterior mud sump location (wash bay drainage interface).
- Detention/retention features if applicable.
- LID elements if applicable.
- Spot elevations at critical drainage points.
- Site boundary and property line reference.

**Step C3 — Produce drainage profiles (if piped storm sewer).**
If a piped storm sewer system is used, produce longitudinal profiles for all storm sewer mains showing: pipe diameter, material, slope, invert elevations at manholes and connections, and crossing clearances with other utilities.

**Step C4 — Produce drainage details and notes.**
Produce standard and project-specific details for: inlet/catch basin construction, outlet structure and energy dissipation, wash bay exterior mud sump (or reference to PKG-018 detail), pipe trench (if applicable), and any non-standard drainage elements. Include drainage legend and general notes on applicable standards and design storm parameters.

**Step C5 — Apply consistent nomenclature.**
Confirm that all terminology, structure names, and elevation datums used on the drainage plan are consistent with DEL-005-02 (Site Grading Plan), DEL-005-05 (Civil Sections and Details), and DEL-005-06 (Calculation Package). Refer to Datasheet Identification table for correct deliverable and project identification information to appear in drawing title blocks.

---

### Phase D — Internal Review and Quality Check

**Step D1 — Internal engineering review.**
The Civil Engineer of record (or designated reviewer) performs a technical review of all drainage plan sheets against:
- Specification requirements REQ-001 through REQ-008.
- Calculation Package (DEL-005-06) consistency.
- Site Grading Plan (DEL-005-02) consistency.
- RFP requirements R-01, §3.3.2 and §3.4.

Document review comments and resolve before IFC release.

**Step D2 — Coordination check with other disciplines.**
Confirm that the drainage plan does not conflict with:
- Building footprint and entrance/exit locations (PKG-001 Architectural).
- Wash bay exterior mud sump drainage detail (PKG-018).
- Utility tie-in locations (SOW-0080, SOW-0081, SOW-0082): perform explicit **utility crossing/conflict check** — verify vertical and horizontal clearance between drainage structures (catch basins, pipe inverts, swale inverts) and utility crossings (water, sewer, electrical). Document any crossing locations and confirm adequate separation distances per applicable code. (Enrichment: X-004 — passive reference to utility routing is insufficient for exhaustive verification scope.)
- Plumbing floor drain and sump drain locations within the building (PKG-006) — confirm no interference at building connections.

**Step D3 — Resolve flagged conflicts.**
Address all items from the Guidance Conflict Table (C-001, C-002, C-003) before IFC drawing release. Document resolutions in the Calculation Package or in drawing general notes.

---

### Phase E — IFC Release and Handoff

**Step E1 — Civil Engineer P.Eng. stamp and signature.**
The Civil Engineer of record applies their P.Eng. stamp and signature to all IFC drawing sheets in accordance with the requirements of R-01, §3.3.2 and SOW-0018. Confirm that the P.Eng. is licensed in the Province of Alberta.

**Step E2 — County preliminary design approval confirmed.**
Confirm that the County has approved the preliminary civil design (DEL-005-01) before releasing this drawing set as IFC. Retain documentation of County approval in project records.

**Step E3 — Assemble regulatory submission package (A-004).**
Prepare the regulatory/permit submission package referencing:
- Specification REQ-006 (code and regulatory compliance) and applicable code clauses.
- Datasheet Conditions (permit requirements, code compliance, inspection coordination).
- All applicable permit application forms and supporting documentation as required by Camrose County and Alberta Safety Codes.
Coordinate with PKG-009 Permitting for submission logistics. (Enrichment: A-004 — regulatory audit readiness requires an explicit step for permit submission package assembly, distinct from IFC distribution.)

**Step E4 — Distribute IFC drawing set.**
Distribute the IFC drainage plan drawing set to:
- County project representative (for records and permit authority reference).
- Construction team responsible for PKG-018 (Sitework and Civil Construction).
- Project Manager for overall coordination.
- Permit authority as required for building permit / development permit applications (coordinated through PKG-009 Permitting).

**Step E5 — Owner/County review and acceptance (D-002).**
Submit the IFC drainage plan drawing set to the Owner (Camrose County) for review and formal acceptance. This step is distinct from the preliminary design approval of DEL-005-01 (which is a prerequisite); this is acceptance of the final DEL-005-03 IFC set. Document Owner review comments and formal acceptance in project records. (Enrichment: D-002 — final value judgment on the drainage plan deliverable requires an Owner acceptance step.)

**Step E6 — Incorporate permit authority review comments (E-003).**
If the permit authority requests revisions after submission (Step E3), receive and review the comments, incorporate required changes into the drainage plan, and re-issue revised drawings through the approval workflow (Steps E1 through E4 as applicable). Document all permit authority comments, responses, and resulting revisions in the project record. (Enrichment: E-003 — comprehensive audit intelligence requires a closed-loop process for regulatory review comments.)

**Step E7 — Record IFC drawing issuance.**
Record the IFC issue date, revision number, and distribution in the project document register. Retain a signed/stamped copy of the IFC drawing set in project records.
