# Specification — DEL-002-01 Preliminary Structural Design

---

## 1. Scope

### 1.1 What This Deliverable Covers

DEL-002-01 is the **Preliminary Structural Design** for the West Dried Meat Lake Regional Landfill (WDMLRL) Maintenance Shop Addition. It is a Design Presentation deliverable produced by the Structural Engineer for review and approval by Camrose County (the Owner) before final structural design proceeds.

The preliminary structural design encompasses:

- Overall structural system concept for the New North Shop Addition (~13,000 sq ft, concrete structure, 35 ft ceiling height)
- Conceptual foundation approach (with explicit acknowledgment of variable-price / post-geotech basis)
- Structural framing and load path concept for the main building
- Conceptual design for the overhead crane runway support structure (two 5-tonne cranes on trolley)
- Conceptual mezzanine framing (load-bearing, above parts room, utility room, and wash bay)
- Conceptual service pit / trench structure (below-grade, ventilated and lighted)
- Conceptual wash bay enclosure structural integration
- Conceptual steel plate embedment locations in concrete floor
- Conceptual stair structure to mezzanine
- Identification of structural elements requiring inter-discipline coordination (architectural, mechanical, electrical, plumbing, civil)
- Any structural implications of Old North Shop renovation (mezzanine, washroom, locker/change room) — to be confirmed in scope at preliminary stage

### 1.2 What This Deliverable Excludes

- Final (IFC-level) structural drawings and details — those are DEL-002-02 through DEL-002-09
- Structural calculation package — DEL-002-10
- Foundation Design Package (variable-price) — DEL-002-11
- Structural Specification — DEL-002-12
- All non-structural disciplines (architectural, mechanical, electrical, civil, plumbing, landscape)
- Construction execution

### 5.2 Typical Preliminary Design Presentation Contents (ASSUMPTION — standard engineering practice)

The following are the types of artifacts typically included in a preliminary structural design presentation for a project of this type. Final content is at the Structural Engineer's discretion, subject to County acceptance:

| Artifact | Description |
|---|---|
| Structural system narrative | Written description of the proposed structural system, material choices, load path, and design philosophy |
| Conceptual floor plan | Annotated plan showing primary structural grid, column/wall locations, crane runway, service pit, mezzanine extent, steel plate zones |
| Conceptual building sections | Cross-sections confirming ceiling height (35 ft), mezzanine level, crane clearance |
| Foundation narrative | Description of proposed foundation type and approach; statement of variable-price basis; assumptions used |
| Mezzanine framing concept | Sketch or description of mezzanine structural system and design load basis |
| Crane runway concept | Sketch or description of runway beam and support concept for two 5-tonne cranes |
| Inter-discipline coordination notes | Key structural interfaces and requirements for other disciplines |
| Code and standard identification | List of intended governing codes, editions, and AHJ |
| Preliminary estimate basis | Foundation pricing assumptions and variable-price scope boundary |
