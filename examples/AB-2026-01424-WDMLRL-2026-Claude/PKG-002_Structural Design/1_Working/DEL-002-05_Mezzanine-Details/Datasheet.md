## 4. Construction

### 4.1 Scope of This Drawing Set

This deliverable covers the **mezzanine framing details** for the New North Shop addition. The drawing set is expected to include (ASSUMPTION based on standard structural drawing set practice for mezzanines):

| Sheet Type | Anticipated Content |
|---|---|
| Mezzanine Framing Plan | Plan view showing beam/joist layout, column locations, and dimensions |
| Mezzanine Sections | Cross-sections through the mezzanine structure showing depth, clearances, and connections |
| Framing Connection Details | Beam-to-column, beam-to-wall, and deck attachment details |
| Stair Framing Details | Stair structural design including stringers, treads, landings, and handrail attachment |
| Guardrail Details | Mezzanine edge protection per ABC requirements |
| Anchorage/Embedment Details | Column base plates and connections to concrete structure |
| General Notes | Material specifications, weld/bolt schedules, design basis |
