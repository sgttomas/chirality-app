## Construction

**Test and Inspection Record Content and Format:**

Each test or inspection record typically includes:

| Record Section | Content | Source |
|----------------|---------|--------|
| **Identification** | Record title, record number, date, project name, system/equipment tested | **ASSUMPTION**: Standard record header |
| **Test/Inspection Scope** | Description of system/equipment tested, test procedure reference (NFPA standard, specification section) | **ASSUMPTION**: Test scope definition |
| **Test Equipment** | Test equipment used (gauges, flow meters, multimeters, etc.), calibration dates | **ASSUMPTION**: Test equipment documentation |
| **Test Personnel** | Names and signatures of personnel performing test, witnessing test (Contractor QC, third-party inspector, AHJ, Employer) | **ASSUMPTION**: Test personnel documentation |
| **Test Conditions** | Ambient conditions, system configuration, any deviations from normal test conditions | **ASSUMPTION**: Test condition documentation |
| **Test Procedure** | Step-by-step test procedure followed (may reference standard procedure) | **ASSUMPTION**: Test procedure documentation |
| **Test Data** | Measured values (pressures, flows, times, voltages, currents, etc.), comparison to acceptance criteria | **ASSUMPTION**: Test data recording |
| **Test Results** | Pass/Fail determination, deviations, corrective actions (if failed), re-test results | **ASSUMPTION**: Test result documentation |
| **Acceptance Criteria** | Specified acceptance criteria from NFPA standards, specifications, or design | **ASSUMPTION**: Acceptance criteria reference |
| **Sign-offs** | Contractor QC sign-off, third-party inspector sign-off (if applicable), AHJ sign-off (if applicable), Employer acceptance sign-off | **ASSUMPTION**: Acceptance sign-offs |
| **Attachments** | Test data sheets, photographs, calibration certificates, manufacturer certificates, as-built markups | **ASSUMPTION**: Record attachments |
