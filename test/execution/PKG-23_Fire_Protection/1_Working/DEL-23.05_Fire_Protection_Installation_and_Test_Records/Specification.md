# Specification: DEL-23.05 Fire Protection Installation & Test Records

## Scope

This specification defines the requirements for **Fire Protection Installation & Test Records** within **PKG-23 Fire Protection** for the Canola Oil Transload Facility — Phase 1 Design & Build project.

**Project Context:**
- Location: Fraser Surrey Terminal, 11060 Elevator Road, Surrey, BC
- Employer: DP World Fraser Surrey Inc.
- Contract Type: Design & Build
- Source: Decomposition document Section 1.1

**Deliverable Scope:**

This deliverable provides evidence of completion, inspection, and testing for fire protection systems per Employer's Requirements (ER), including:

**Anticipated deliverable artifacts (test and inspection records):**
- Hydrostatic test records (Source: Decomposition DEL-23.05 anticipated artifacts)
- Flow test records (Source: Decomposition DEL-23.05 anticipated artifacts)
- Fire alarm test records (Source: Decomposition DEL-23.05 anticipated artifacts)

**Additional test and inspection records (typical for comprehensive fire protection commissioning):**
- Fire pump performance test records (if applicable) — **ASSUMPTION**: If fire pump in Contractor scope
- Foam system test records — **ASSUMPTION**: Required for foam system commissioning
- Foam concentrate sample and test certificates — **ASSUMPTION**: Foam quality assurance
- Fire water pipe flushing records — **ASSUMPTION**: Piping commissioning
- Fire hydrant installation and inspection records — **ASSUMPTION**: Hydrant commissioning
- Fire alarm device testing and verification records — **ASSUMPTION**: Device-level testing
- Fire alarm system integration test records — **ASSUMPTION**: SCADA integration testing
- Visual inspections and as-built verification — **ASSUMPTION**: As-built verification
- Valve operation test records — **ASSUMPTION**: Valve commissioning
- Seismic anchorage inspection records — **ASSUMPTION**: BC seismic inspection
- Training records — **ASSUMPTION**: Operations training
- AHJ inspection and approval records — **ASSUMPTION**: Authority approval
- Commissioning reports and certificates — **ASSUMPTION**: Commissioning closeout

**Purpose of Installation and Test Records:**

Installation and test records serve to:
1. Demonstrate fire protection systems installed per design (drawings and specifications)
2. Verify fire protection systems meet performance requirements (from calculations and specifications)
3. Demonstrate code compliance (NFPA, NBCC, BCFC requirements)
4. Obtain AHJ approval and acceptance (fire marshal, building official)
5. Provide evidence for insurance approval and facility operations permit
6. Create permanent facility records for operations, maintenance, and future modifications
7. Document training provided to facility operations personnel
- Source: **ASSUMPTION**: Standard purpose of commissioning and test records

**Scope Boundaries:**

| Included | Excluded / Interface |
|----------|---------------------|
| Testing and inspection records for fire protection systems | Detailed testing procedures (covered in DEL-23.02 specification and NFPA standards) |
| Test witnessing and acceptance sign-offs | Operations and maintenance manuals (separate O&M deliverable) |
| As-built verification and documentation | Ongoing operations and maintenance records (facility responsibility post-handover) |
| Commissioning completion certification | Warranty documentation (separate contract/procurement documentation) |

**Project Objectives Supported:**

This deliverable supports **OBJ-1: Safe & Reliable Operation** — test records verify fire protection systems meet design requirements and are ready for safe facility operations.
- Source: Decomposition Section 2 (Project Objectives) and Section 6 (Objective-to-Deliverable Mapping)

## Requirements

### Functional Requirements

**FR-1: Comprehensive Test and Inspection Coverage**
- Test and inspection records shall cover all fire protection systems and equipment requiring commissioning and acceptance testing
- Testing scope shall include all systems shown on DEL-23.01 drawings and specified in DEL-23.02 specification
- Source: **ASSUMPTION**: Testing completeness requirement
- Verification: Cross-check test records with equipment list and system list from DEL-23.01

**FR-2: Testing Per Code Requirements**
- Testing shall be performed per applicable NFPA standards (NFPA 24, NFPA 291, NFPA 20, NFPA 11, NFPA 72) and Employer's Requirements
- Test procedures shall follow code-prescribed methods
- Source: **ASSUMPTION**: Code compliance requirement for testing
- Verification: Test procedures reference NFPA standards

**FR-3: Performance Verification**
- Test results shall verify fire protection systems meet design performance requirements from DEL-23.03 (flow rates, pressures, capacities, response times)
- Test results compared to acceptance criteria from DEL-23.02 specification and DEL-23.03 calculations
- Source: **ASSUMPTION**: Performance verification requirement
- Verification: Test results compared to design requirements

**FR-4: As-Built Verification**
- Inspections shall verify systems installed per DEL-23.01 drawings and DEL-23.02 specifications
- As-built deviations documented and resolved
- Source: **ASSUMPTION**: As-built verification requirement
- Verification: As-built inspection records and as-built markups

**FR-5: AHJ Inspection and Approval**
- Fire protection systems shall be inspected and approved by authorities having jurisdiction (AHJ) — fire marshal, building official
- AHJ inspection and approval documented
- Source: **ASSUMPTION**: AHJ approval requirement for fire protection systems
- Verification: AHJ inspection reports and approval certificates

**FR-6: Test Equipment Calibration**
- Test equipment (pressure gauges, flow meters, multimeters, etc.) shall be calibrated and calibration current
- Calibration certificates included in test records
- Source: **ASSUMPTION**: Test equipment quality assurance
- Verification: Calibration dates and certificates in test records

**FR-7: Test Witnessing**
- Testing shall be witnessed by Contractor QC and, where required, by third-party inspector, AHJ, or Employer
- Witness sign-offs documented
- Source: **ASSUMPTION**: Test witness requirement
- Verification: Witness sign-offs on test records

**FR-8: Training Documentation**
- Operations training provided to facility personnel documented
- Training records include training content, attendees, date
- Source: **ASSUMPTION**: Operations training requirement
- Verification: Training records and sign-offs

### Performance Requirements

**PR-1: Test Record Accuracy**
- Test records shall accurately document test procedures, test data, and test results
- Test data traceable and reproducible
- Source: **ASSUMPTION**: Test record accuracy requirement

- Test and inspection records sufficiently complete to demonstrate compliance with design, specifications, and codes

- Source: **ASSUMPTION**: Record completeness requirement
- Verification: Record review for completeness

**PR-4: Test Record Consistency**
- Test results shall be consistent with design calculations (DEL-23.03), specifications (DEL-23.02), and data sheets (DEL-23.04)
- Significant deviations investigated and resolved
- Source: **ASSUMPTION**: Test-design consistency requirement
- Verification: Comparison of test results to design requirements

**PR-5: Commissioning Completion**
- All testing and inspections completed prior to system acceptance and operations turnover
- Commissioning completion certified
- Source: **ASSUMPTION**: Commissioning completion requirement
- Verification: Commissioning completion certificate

## Verification

**Verification Methods for Test and Inspection Records:**

**V-1: Test Performance Verification**
- Qualified personnel perform testing per code procedures and specifications
- Test equipment calibrated and appropriate for measurements
- Test witnessed by required parties (Contractor QC, third-party QC, AHJ, Employer)
- Verification record: Completed test forms with witness sign-offs
- Source: **ASSUMPTION**: Test performance verification

**V-2: Test Result Review**
- Test results reviewed for accuracy, completeness, and pass/fail determination
- Test results compared to acceptance criteria from specifications and calculations
- Verification record: Test record review sign-off
- Source: **ASSUMPTION**: Test result verification

**V-3: Failed Test Resolution**
- Failed tests result in corrective actions, system corrections, and re-testing
- Re-test results documented
- Verification record: Corrective action documentation and re-test records
- Source: **ASSUMPTION**: Failed test resolution process

**V-4: Code Compliance Verification**
- Test records reviewed for code compliance demonstration (NFPA, NBCC, BCFC)
- Verification record: Code compliance review sign-off
- Source: **ASSUMPTION**: Code compliance verification

**V-5: AHJ Inspection and Approval**
- AHJ inspects fire protection systems and reviews test records
- AHJ issues approval or permit
- Verification record: AHJ inspection report and approval certificate
- Source: **ASSUMPTION**: AHJ approval verification

**V-6: Commissioning Completion Verification**
- All test and inspection records reviewed for completeness
- Commissioning completion certified
- Verification record: Commissioning completion certificate
- Source: **ASSUMPTION**: Commissioning closeout verification

**Acceptance Criteria:**

Test and inspection records are acceptable when:
- All required testing and inspections completed
- All test results pass acceptance criteria (or failed tests corrected and re-tested to pass)
- All required witness sign-offs obtained (Contractor QC, third-party QC, AHJ, Employer)
- Test records complete and accurate
- AHJ inspection and approval obtained
- Commissioning completion certified
- Source: **ASSUMPTION**: Test record acceptance criteria — specific criteria **TBD** per project
