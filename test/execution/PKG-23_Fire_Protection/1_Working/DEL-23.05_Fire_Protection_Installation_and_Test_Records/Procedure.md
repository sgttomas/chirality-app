# Procedure: DEL-23.05 Fire Protection Installation & Test Records

## Purpose

This procedure defines the process for **producing** the **Fire Protection Installation & Test Records** within **PKG-23 Fire Protection** for the Canola Oil Transload Facility — Phase 1 Design & Build project.

**Scope of This Procedure:**

This procedure covers the workflow for planning, performing, witnessing, documenting, and approving fire protection system testing and inspections per project quality requirements and applicable codes.

**Deliverable Information:**
- **Deliverable ID:** DEL-23.05
- **Deliverable Type:** Record
- **Responsible Party:** D&B Contractor (QA/QC)
- **Discipline:** Specialty (Fire Protection)
- **Source:** Decomposition DEL-23.05 and `_CONTEXT.md`

**Note on Procedure Interpretation:**

This Procedure document is **dual-use**:
1. **Production procedure:** How to produce the test record package (planning testing, performing tests, documenting results)
2. **Testing procedure:** The actual testing and inspection procedures to execute (detailed test steps)

This document emphasizes the **production procedure** (how to create the deliverable). Detailed testing procedures are prescribed by NFPA codes, equipment manufacturer instructions, and project specifications (DEL-23.02).

## Steps

### Step 1: Testing Planning and Preparation

**1.1 Develop Test Plan**
- Review fire protection design documents (DEL-23.01, DEL-23.02, DEL-23.03, DEL-23.04) to understand testing scope
- Review applicable NFPA testing standards (NFPA 24, NFPA 291, NFPA 20, NFPA 11, NFPA 72)
- Develop comprehensive test plan listing:
  - All systems and equipment requiring testing
  - Test procedures to be followed (NFPA standards, specifications)
  - Acceptance criteria (from DEL-23.02 and DEL-23.03)
  - Test sequence and schedule
  - Test personnel and witness requirements
  - Test equipment requirements
- **Output:** Fire protection testing plan (document or checklist)

**1.2 Obtain Test Forms and Templates**
- Obtain or develop test form templates for each test type:
  - Hydrostatic test form
  - Flow test form
  - Fire pump performance test form
  - Foam system test form
  - Fire alarm device test form
  - Fire alarm system test form
  - Visual inspection checklists
  - Valve operation test form
- **TBD:** Project test form templates
- **Output:** Test forms ready for use

**1.3 Procure and Calibrate Test Equipment**
- Procure test equipment (pressure gauges, pitot gauge, flow meters, multimeters, etc.)
- Calibrate test equipment at accredited calibration laboratory
- Obtain and file calibration certificates
- **Output:** Calibrated test equipment, calibration certificates

**1.4 Coordinate Witnesses**
- Identify witness requirements for each test:
  - Contractor QC: witnesses all tests
  - Third-party QC: witnesses critical tests (if required)
  - AHJ: witnesses fire alarm testing and other tests as required by AHJ
  - Employer: witnesses tests if required by contract
- Notify witnesses of testing schedule
- Confirm witness availability
- **Output:** Witness coordination and schedule

**1.5 Verify System Readiness for Testing**
- Verify fire protection systems installation complete and ready for testing:
  - Fire water piping installed and ready for hydrostatic test (before burial for underground pipe)
  - Fire hydrants installed and ready for flow test
  - Fire pump installed and ready for performance test (if applicable)
  - Foam systems installed and ready for proportioning test
  - Fire alarm system installed, wired, programmed, and ready for functional test
- Resolve any installation deficiencies before testing
- **Output:** System readiness verification

### Step 2: Fire Water Piping Hydrostatic Testing

**2.1 Prepare Fire Water Piping for Hydrostatic Test**
- Fill fire water piping with water
- Vent air from system (high points vented)
- Isolate test section (close valves to isolate section to be tested)
- Connect test pump and pressure gauges
- **Test Procedure Reference:** NFPA 24 Section 10.3 Hydrostatic Test
- **Output:** Fire water piping prepared for hydrostatic test

**2.2 Perform Hydrostatic Test**
- Pressurize fire water piping to test pressure:
  - Test pressure = 200 psi or working pressure + 50 psi, whichever is greater (per NFPA 24)
  - Working pressure from DEL-23.03 hydraulic calculation — **TBD**
- Maintain test pressure for 2 hours (per NFPA 24)
- Monitor pressure (record initial pressure, periodic readings, final pressure)
- Inspect piping for leaks during test
- **Acceptance Criteria:** No leaks; pressure drop not exceeding **TBD** psi over 2-hour duration (typically ≤5 psi for piping system per NFPA 24 interpretation)
- **Test Data:** Record initial pressure, time, final pressure, pressure drop, leak observations
- **Output:** Hydrostatic test data

**2.3 Document Hydrostatic Test Results**
- Complete hydrostatic test record form:
  - Test date, test personnel, witnesses
  - Test section description (piping from valve X to valve Y)
  - Test pressure, duration
  - Test data (initial/final pressures, pressure drop)
  - Leak observations (none / location and description)
  - Pass/fail determination
  - Witness sign-offs (Contractor QC, third-party QC if required)
- Photograph piping during test (if practical)
- **Output:** Completed hydrostatic test record

**2.4 Corrective Actions and Re-Test (if test failed)**
- If test failed (leaks or excessive pressure drop):
  - Identify and repair leaks
  - Depressurize, drain, repair, refill
  - Re-test per procedure above
  - Document corrective actions and re-test results
- **Output:** Corrective action documentation and re-test records (if applicable)

### Step 3: Fire Hydrant Flow Testing

**3.1 Prepare for Flow Test**
- Identify hydrants to be tested (typically most hydraulically remote hydrant, or representative sample, or all hydrants per project requirements)
- Identify pressure measurement location (typically hydrant nearest water supply source)
- Obtain flow test equipment (pitot gauge, pressure gauges, calibrated)
- Coordinate witnesses (Contractor QC, third-party QC if required, AHJ if required)
- **Test Procedure Reference:** NFPA 291 Fire Flow Testing
- **Output:** Flow test preparation complete

**3.2 Perform Flow Test**
- Measure static pressure at pressure measurement hydrant (all hydrants closed)
- Open flowing hydrant and allow flow to stabilize
- Measure residual pressure at pressure measurement hydrant (while flowing hydrant open)
- Measure pitot pressure at flowing hydrant discharge nozzle
- Calculate flow rate from pitot pressure per NFPA 291 (Q = 29.83 × d² × √P, where d = nozzle diameter in inches, P = pitot pressure in psi)
- **Test Data:** Record static pressure, residual pressure, pitot pressure, calculated flow rate, nozzle size
- **Acceptance Criteria:** Flow rate ≥ **TBD** gpm at residual pressure ≥ **TBD** psi (per DEL-23.03 hydraulic calculation)
- **Output:** Flow test data

**3.3 Document Flow Test Results**
- Complete flow test record form for each hydrant tested:
  - Test date, test personnel, witnesses
  - Flowing hydrant tag (e.g., FH-005)
  - Pressure measurement hydrant tag (e.g., FH-001)
  - Static pressure, residual pressure, pitot pressure
  - Nozzle diameter, calculated flow rate
  - Pass/fail determination (compare to acceptance criteria)
  - Witness sign-offs
- Photograph hydrant during flow test (flowing hydrant with pitot gauge)
- **Output:** Completed flow test records

### Step 4: Fire Pump Performance Testing (if applicable)

**4.1 Prepare for Fire Pump Test**
- Verify fire pump installation complete and ready for testing
- Verify fire pump test header or discharge connection available for flow control
- Obtain test equipment (flow meter, pressure gauges, tachometer, ammeter/voltmeter, all calibrated)
- Coordinate witnesses (Contractor QC, third-party QC, fire pump manufacturer technician, AHJ if required)
- **Test Procedure Reference:** NFPA 20 Chapter 14 Pump Acceptance Test
- **Output:** Fire pump test preparation complete

**4.2 Perform Fire Pump Performance Test**
- Start fire pump
- Test pump at multiple flow rates per NFPA 20:
  - 0% flow (churn condition)
  - 100% rated flow
  - 150% rated flow
- At each test point, measure:
  - Flow rate (gpm)
  - Suction pressure (psi)
  - Discharge pressure (psi)
  - Pump speed (rpm)
  - Motor current and voltage (if electric motor)
- Calculate pump head: Head (psi) = Discharge pressure – Suction pressure (+ elevation correction if applicable)
- Compare measured pump performance to manufacturer pump curve and design requirements (from DEL-23.03)
- **Acceptance Criteria:** Pump performance meets manufacturer pump curve within NFPA 20 tolerances; pump meets design capacity **TBD** gpm at **TBD** psi
- **Output:** Fire pump test data

**4.3 Document Fire Pump Test Results**
- Complete fire pump performance test record:
  - Test date, test personnel, witnesses
  - Fire pump tag (e.g., FP-001)
  - Test data table (flow, pressures, head, speed, power at each test point)
  - Pump curve comparison (actual performance vs. manufacturer curve)
  - Pass/fail determination
  - Witness sign-offs
- Attach manufacturer pump curve and test data plot
- **Output:** Completed fire pump test record

### Step 5: Foam System Testing

**5.1 Prepare for Foam System Test**
- Verify foam system installation complete (foam concentrate storage tank, proportioning system, foam piping, discharge devices)
- Verify foam concentrate procured, delivered, and filled in storage tank
- Obtain foam test equipment (refractometer or titration kit for proportioning test, containers for foam collection, graduated cylinders for foam quality test)
- Coordinate witnesses (Contractor QC, third-party QC if required)
- **Test Procedure Reference:** NFPA 11 Section 7.7 System Acceptance Tests
- **Output:** Foam system test preparation complete

**5.2 Perform Foam Proportioning Accuracy Test**
- Operate foam proportioning system
- Collect foam solution samples at discharge points
- Measure foam concentrate percentage using refractometer or titration method per NFPA 11
- **Acceptance Criteria:** Foam concentrate percentage within ±10% of nominal (e.g., 3% ± 0.3% or 6% ± 0.6%) per NFPA 11 and foam concentrate manufacturer
- **Output:** Foam proportioning test data

**5.3 Perform Foam Discharge Rate Test**
- Operate foam system and measure foam solution discharge rate (gpm)
- Compare to design discharge rate from DEL-23.03 foam system calculations
- **Acceptance Criteria:** Discharge rate ±10% of design rate (per NFPA 11 and design requirements)
- **Output:** Foam discharge rate test data

**5.4 Perform Foam Quality Test (if required)**
- Collect foam discharge sample in graduated container
- Measure foam expansion ratio (volume of foam / volume of foam solution)
- Measure 25% drainage time (time for 25% of foam solution to drain from foam)
- Compare to foam concentrate listing requirements and NFPA 11 guidelines
- **Acceptance Criteria:** Foam quality meets foam concentrate UL listing requirements
- **Output:** Foam quality test data

**5.5 Document Foam System Test Results**
- Complete foam system test record:
  - Test date, test personnel, witnesses
  - Foam system identification (e.g., Tank T-001 foam system)
  - Foam concentrate type and percentage
  - Proportioning accuracy test results
  - Discharge rate test results
  - Foam quality test results (if performed)
  - Pass/fail determination
  - Witness sign-offs
- Attach foam concentrate certificates (UL listing, manufacturer certificate of compliance)
- **Output:** Completed foam system test records

### Step 6: Fire Alarm System Testing

**6.1 Prepare for Fire Alarm Testing**
- Verify fire alarm system installation complete (control panel, detection devices, notification devices, wiring)
- Verify fire alarm system programmed and operational
- Coordinate fire alarm manufacturer commissioning technician to perform/support testing
- Coordinate witnesses (Contractor QC, AHJ fire marshal)
- **Test Procedure Reference:** NFPA 72 Chapter 14 Inspection, Testing, and Maintenance
- **Output:** Fire alarm test preparation complete

**6.2 Perform Fire Alarm Device Testing**
- Test each detection device (heat detectors, flame detectors, smoke detectors, manual pull stations):
  - Activate device using appropriate test method (heat source for heat detector, test aerosol for smoke detector, flame simulator for flame detector, manually activate pull station)
  - Verify alarm signal reaches control panel (addressable device ID displayed)
  - Verify notification devices activate (horns sound, strobes flash)
  - Verify alarm annunciated correctly at control panel
- Document test results for each device on fire alarm device test form or checklist
- **Acceptance Criteria:** All devices functional; alarm signal reaches panel within 10 seconds (per NFPA 72); notification devices activate correctly
- **Output:** Fire alarm device test data (device-by-device test results)

**6.3 Perform Fire Alarm Notification Device Testing**
- Test each notification device (horns, strobes):
  - Activate alarm zone
  - Verify horns audible (measure decibel level if required)
  - Verify strobes visible (measure candela if required)
- Verify notification device coverage per NFPA 72 (audibility and visibility throughout protected areas)
- **Acceptance Criteria:** Notification devices meet NFPA 72 audibility and visibility requirements
- **Output:** Fire alarm notification device test data

**6.4 Perform Fire Alarm Battery Load Test**
- Disconnect fire alarm panel from normal power supply (simulate power failure)
- Operate on battery backup
- Verify system operates on battery for required duration (24-hour standby + 5-minute alarm per NFPA 72, or per DEL-23.03 calculation)
- Monitor battery voltage during test
- **Acceptance Criteria:** Battery capacity meets NFPA 72 requirements (24 hours standby + 5 minutes alarm minimum)
- **Output:** Fire alarm battery test data

**6.5 Perform Fire Alarm System Integration Test (if applicable)**
- Test fire alarm system integration with facility SCADA/DCS (coordination with PKG-19 I&C team)
- Verify alarm signals transmitted to SCADA/DCS
- Verify SCADA/DCS displays alarm information correctly
- Verify SCADA/DCS control functions (if applicable, e.g., acknowledge alarms)
- **Acceptance Criteria:** Fire alarm system integrates correctly with SCADA/DCS; alarm signals transmitted and displayed correctly
- **Output:** Fire alarm integration test data

**6.6 Document Fire Alarm Test Results**
- Complete fire alarm test records:
  - Fire alarm device test form (list of all devices tested, results)
  - Fire alarm notification device test form
  - Fire alarm battery test form
  - Fire alarm integration test form (if applicable)
  - Overall fire alarm system test summary
- Include witness sign-offs (Contractor QC, fire alarm technician, AHJ fire marshal)
- **Output:** Completed fire alarm test records

### Step 7: Additional Testing and Inspections

**7.1 Fire Water Pipe Flushing (prior to hydrostatic test or after)**
- Flush fire water piping to remove construction debris
- Operate flushing at velocity ≥10 fps or per NFPA 24
- Continue flushing until water runs clear
- Document flushing (section flushed, duration, water quality at completion)
- **Output:** Pipe flushing records

**7.2 Fire Hydrant Installation Inspection**
- Inspect fire hydrant installations:
  - Hydrant plumb and properly positioned
  - Concrete pad installed
  - Shutoff valve accessible in valve box
  - Drain valve functional
  - Hydrant operates smoothly
- Document inspection on checklist or form
- **Output:** Fire hydrant installation inspection records

**7.3 Valve Operation Testing**
- Test operation of fire water isolation valves and control valves:
  - Operate valve open and closed
  - Verify valve operates smoothly
  - Verify valve position indication (if applicable)
- Document valve operation test results
- **Output:** Valve operation test records

**7.4 Visual Inspection and As-Built Verification**
- Perform visual inspection of all fire protection installations:
  - Verify installations per drawings (DEL-23.01) and specifications (DEL-23.02)
  - Document as-built deviations (if any) with photographs and markups
  - Verify equipment tags match drawings
  - Verify system configuration per design
- Complete visual inspection checklists
- **Output:** Visual inspection records, as-built deviation documentation (if any)

**7.5 Seismic Anchorage Inspection**
- Inspect seismic anchorage for fire protection equipment and piping supports:
  - Verify anchorage installed per structural design (coordinate with PKG-06 structural team)
  - Verify anchorage meets NBCC 2020 seismic requirements
- Document seismic anchorage inspection
- **Output:** Seismic anchorage inspection records

### Step 8: AHJ Inspection and Approval

**8.1 Coordinate AHJ Inspection**
- Schedule AHJ inspection with fire marshal or building official
- Provide AHJ with advance notice and documentation (drawings, specifications, test plan)
- Coordinate AHJ witness of testing (fire alarm testing typically witnessed by fire marshal)
- **Output:** AHJ inspection scheduled

**8.2 AHJ Inspection**
- AHJ inspector performs inspection of fire protection installations
- AHJ may witness testing or review test records
- Address any AHJ comments or deficiencies
- **Output:** AHJ inspection performed

**8.3 Obtain AHJ Approval**
- Obtain AHJ inspection report and approval certificate or permit
- File AHJ approval documents
- **Output:** AHJ approval certificate or permit

### Step 9: Training

**9.1 Conduct Operations Training**
- Provide fire protection system training to facility operations personnel:
  - Fire water system operation (valves, isolation, fire pump startup if applicable)
  - Fire hydrant operation
  - Foam system operation (if operator-initiated)
  - Fire alarm system operation (alarm acknowledgment, reset, manual activation)
  - Periodic testing requirements (fire alarm monthly testing, flow test annually, etc.)
  - Emergency response procedures
- Training provided by commissioning engineer, fire alarm technician, or fire pump technician
- **TBD:** Training requirements and content per Employer or contract
- **Output:** Training provided to operations personnel

**9.2 Document Training**
- Complete training records:
  - Training date, training content, trainer
  - Training attendees (names, signatures)
  - Training materials provided (manuals, procedures)
- **Output:** Training records

### Step 10: Commissioning Completion and Record Package Assembly

**10.1 Review Test and Inspection Records for Completeness**
- Review all test and inspection records:
  - All required testing completed
  - All test results pass (or failures corrected and re-tested to pass)
  - All witness sign-offs obtained
  - All documentation complete
- Identify any missing records or incomplete testing
- Complete any remaining testing or documentation
- **Output:** Complete test and inspection record set

**10.2 Prepare Commissioning Completion Certificate**
- Prepare fire protection commissioning completion certificate:
  - Systems commissioned: fire water distribution, fire hydrants, fire pump (if applicable), foam systems, fire alarm system
  - Testing completed per NFPA standards and specifications
  - All tests passed
  - AHJ approval obtained
  - Training provided
  - Systems ready for operations
  - Commissioning engineer certification and signature
- **Output:** Commissioning completion certificate

**10.3 Assemble Test Record Package**
- Assemble all test and inspection records into organized package:
  - Cover sheet with record package title, deliverable ID, date
  - Table of contents
  - Test and inspection records organized by system type
  - AHJ inspection and approval records
  - Training records
  - Commissioning completion certificate
  - Calibration certificates (appendix)
- Prepare electronic record package (PDFs) and/or hard copy binders
- **Output:** Complete test record package

**10.4 Review and Approval**
- QC manager or commissioning manager reviews test record package for completeness and quality
- Discipline lead or project manager approves test record package
- **Output:** Approved test record package

**10.5 Issue Test Record Package**
- Issue test record package via project document management system or per document control procedures
- Provide copies to: Employer/Owner (permanent facility records), project files, AHJ (if required)
- **Output:** Issued test record package, transmittal record

**10.6 Update Deliverable Status**
- Update `_STATUS.md` to reflect deliverable lifecycle state:
  - INITIALIZED → IN_PROGRESS (when testing planning commences)
  - IN_PROGRESS → CHECKING (when test records under review)
  - CHECKING → ISSUED (when test records accepted and issued)
- Copy issued test records to `3_Issued/` folder per framework conventions
- **Output:** Updated `_STATUS.md`, issued test records archived in `3_Issued/`

**10.7 Transfer Records to Employer/Owner**
- Transfer test and inspection records to Employer/Owner at project completion (turnover)
- Obtain Employer/Owner receipt acknowledgment
- Records become permanent facility records under Owner custody
- **Output:** Records transferred to Owner, transfer receipt
