# Procedure: DEL-31.05 Vendor Documentation

## Purpose

To collect, review, organize, and deliver vendor-supplied technical documentation for the Canola Oil Transload Facility Phase 1.

**Source:** Decomposition DEL-31.05 (line 690)

**Deliverable type:** Document Package
**Responsible party:** D&B Contractor

## Steps

### Step 1: Planning and Specification

**Activities:**
1.1. Identify all equipment requiring vendor documentation
1.2. Specify vendor documentation requirements in purchase orders/specifications (manuals, certificates, warranties, spare parts catalogs, format, language)
1.3. Establish vendor documentation submittal and review procedures
1.4. Create vendor documentation register/tracking log

**Deliverables:** Vendor documentation requirements specified; tracking register established

### Step 2: Vendor Coordination and Submittal

**Activities:**
2.1. Coordinate with vendors to confirm documentation requirements and submittal schedules
2.2. Receive vendor documentation submittals
2.3. Log submittals in vendor documentation register

**Deliverables:** Vendor documentation submittals received and logged

### Step 3: Review and Approval

**Activities:**
3.1. Review vendor submittals for completeness (all required documents included)
3.2. Review for content adequacy (manuals comprehensive, certificates complete, warranties valid)
3.3. Review for quality (legible, accurate, correct equipment model/configuration)
3.4. Reject/request resubmittal if inadequate; notify vendor of deficiencies
3.5. Approve vendor documentation upon satisfactory review
3.6. Update vendor documentation register with review status

**Deliverables:** Vendor documentation reviewed and approved; register updated

### Step 4: Organization and Indexing

**Activities:**
4.1. Organize approved vendor documentation by equipment or system
4.2. Create vendor documentation packages (group related documents)
4.3. Develop vendor documentation index with equipment tags, vendor names, document titles, file locations
4.4. File electronic copies in document management system
4.5. File hard copies (if applicable) in organized binders or filing system

**Deliverables:** Vendor documentation organized; index complete

### Step 5: Cross-Referencing and Integration

**Activities:**
5.1. Cross-reference vendor operation manuals to Operation Manuals (DEL-31.03)
5.2. Cross-reference vendor maintenance manuals to Maintenance Manuals (DEL-31.04)
5.3. Cross-reference vendor warranties to Guarantees & Warranties Register (DEL-31.08) and Warranty Certificates (DEL-31.09)
5.4. Provide vendor documentation access to operations and maintenance manual development teams

**Deliverables:** Cross-references established; vendor documentation accessible to manual development teams

### Step 6: Completeness Verification and Handover

**Activities:**
6.1. Verify all equipment vendor packages obtained and approved (cross-check against equipment list)
6.2. Verify vendor documentation index complete
6.3. Employer acceptance (if required per contract)
6.4. Handover vendor documentation package to Employer
6.5. Provide copies to operations and maintenance teams

**Deliverables:** Complete Vendor Documentation Package; handover complete

### Step 7: Configuration Management

**Activities:**
7.1. Establish process for managing vendor documentation updates (vendor revisions, equipment modifications)
7.2. Assign responsibility for vendor documentation maintenance during operations

**Deliverables:** Configuration management process established

## Verification

1. **Completeness Review** (Step 6.1) — All equipment vendor packages obtained
2. **Content Review** (Step 3.2) — Manuals, certificates, warranties, spare parts lists included
3. **Quality Review** (Step 3.3) — Documentation legible and accurate
4. **Index Verification** (Step 6.2) — Index complete and accurate

**Acceptance Criteria:** All vendor packages complete and approved; index complete; Employer acceptance obtained (if required)

## Records

**Outputs:**
1. Vendor Documentation Packages (organized by equipment/system)
2. Vendor Documentation Index/Register
3. Vendor Submittal and Review Log

**Management:** Vendor documentation filed in `3_Issued/` or package-specific folders; electronic copies in document management system

**Retention:** For facility operational life
