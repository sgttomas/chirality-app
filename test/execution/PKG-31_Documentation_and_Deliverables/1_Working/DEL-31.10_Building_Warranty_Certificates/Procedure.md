# Procedure: DEL-31.10 Building Warranty Certificates

## Purpose

To collect and deliver Building Warranty Certificates for the Canola Oil Transload Facility Phase 1.

**Source:** Decomposition DEL-31.10 (line 696)

**Deliverable type:** Document Package
**Responsible party:** D&B Contractor

## Prerequisites

### Dependencies

**Dependency Tracking:** See \`_DEPENDENCIES.md\` — **NOT_TRACKED**

**Upstream Dependencies:**
- Building construction completion
- Building contractor/supplier warranties

**Source:** **ASSUMPTION**

## Steps

### Step 1: Planning

**Activities:**
1.1. Identify all applicable building warranties (modular building, envelope, roofing, structural)

**Deliverables:** Building warranty list

### Step 2: Certificate Collection

**Activities:**
2.1. Coordinate with building contractors/suppliers to obtain warranty certificates
2.2. Ensure warranty registration completed (if required)
2.3. Verify certificates are originals

**Deliverables:** Building warranty certificates collected

### Step 3: Package Organization

**Activities:**
3.1. Organize certificates by building system
3.2. Cross-reference to Guarantees & Warranties Register (DEL-31.08)

**Deliverables:** Organized Package

### Step 4: Review and Approval

**Activities:**
4.1. Completeness review
4.2. Authenticity check
4.3. Employer acceptance
4.4. Final approval

**Deliverables:** Approved Package

### Step 5: Issuance and Handover

**Activities:**
5.1. Issue Package
5.2. Handover to Employer

**Deliverables:** Issued Package; handover complete

## Verification

1. **Completeness Review** — All building warranties obtained
2. **Authenticity Check** — Certificates are originals

**Acceptance Criteria:** All certificates obtained; authentic; Employer acceptance obtained

## Records

**Outputs:**
1. Building Warranty Certificate Package

**Management:** \`3_Issued/\`

**Retention:** For facility operational life (long-term for building warranties)
