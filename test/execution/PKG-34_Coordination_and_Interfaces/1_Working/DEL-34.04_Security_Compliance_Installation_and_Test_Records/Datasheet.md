## Conditions

**Operating / Security Context:**

This deliverable provides evidence of security compliance activities for construction at Fraser Surrey Terminal, a secure port facility subject to CBSA (border security) and Transport Canada (marine security) regulations.

**Port Security Context:**
- **Location:** Fraser Surrey Terminal, Fraser River, Surrey, BC — designated secure port facility
- **Facility classification:** Marine port facility subject to Marine Transportation Security Regulations (MTSR)
- **Border security:** Facility within CBSA-controlled area; border security protocols apply
- **Objective alignment:** OBJ-5 "Terminal Continuity" — maintain secure facility operations during construction (Decomposition line 63)

**Regulatory Framework:**
- **CBSA (Canada Border Services Agency):** Border security, customs clearance, restricted area access control
- **Transport Canada:** Marine Transportation Security Regulations (MTSR), Facility Security Plans, security threat assessments
- **Facility Security Plan:** Fraser Surrey Terminal maintains approved Facility Security Plan per MTSR — construction activities must comply
- **Security levels:** MARSEC (Maritime Security) levels 1-3 (normal, heightened, exceptional threat)

**Security Compliance Topics:**
- Personnel security clearances (background checks, restricted area access authorization)
- Site access control (badge issuance, visitor management, vehicle access)
- Restricted area access (marine restricted zones, secure cargo areas)
- Security incidents and investigations (unauthorized access, security breaches)
- Security equipment and infrastructure (fencing, gates, cameras, lighting, alarms)
- Security training and awareness (personnel security briefings, threat recognition)
- CBSA and Transport Canada inspections and audits

**Phase/Timing:**
- Security compliance required from construction mobilization through to project handover
- Personnel security clearances obtained prior to site access
- **TBD** — Timeline for security clearance applications and approvals (typically 4-8 weeks)

**Record Types and Structure:**

### 1. Personnel Security Clearance Records

**Purpose:** Document personnel security clearances for restricted area access.

**Typical Content:**
- Personnel roster (name, employer, role, clearance level)
- Security clearance applications (submitted to CBSA/TC)
- Security clearance approvals (CBSA/TC authorization letters or certificates)
- Security clearance expiry dates and renewals
- Security training records (security briefings, awareness training)
- Security incident history (if applicable)

### 2. Site Access Control Records

**Purpose:** Document site access control measures and compliance.

**Typical Content:**
- Site access badges issued (badge number, personnel name, employer, issue date, expiry date)
- Visitor logs (date, name, company, purpose, escort, time in/out)
- Vehicle access permits (vehicle make/model/plate, driver, authorization)
- Restricted area access logs (who accessed, when, purpose, authorization)
- Lost or stolen badge reports and deactivations

### 3. Security Incident Reports

**Purpose:** Document security incidents and investigations.

**Typical Content:**
- Incident date, time, location, description
- Personnel involved (if applicable)
- Witness statements
- Immediate actions taken (site secured, CBSA/TC notified)
- Investigation findings and corrective actions
- Regulatory notifications (CBSA, Transport Canada, law enforcement)

### 4. Regulatory Inspections and Audits

**Purpose:** Document CBSA and Transport Canada security inspections and audit results.

**Typical Content:**
- Inspection/audit date and inspector/auditor name
- Scope of inspection/audit (personnel clearances, site access control, security infrastructure)
- Findings and non-conformances (if any)
- Corrective action plans and implementation records
- Follow-up inspections and closeout

### 5. Security Coordination Records

**Purpose:** Document coordination with CBSA, Transport Canada, and Terminal security.

**Typical Content:**
- Security coordination meeting minutes
- Correspondence with CBSA and Transport Canada
- Facility Security Plan compliance checklists
- MARSEC level changes and notifications
- Security threat assessments and responses

**Data Sources:**
- CBSA and Transport Canada security clearance approvals
- Site access control system (badge issuance, visitor logs)
- Security incident reports (from D&B Contractor security coordinator or site manager)
- Regulatory inspection reports (from CBSA, Transport Canada)
- Security coordination meetings and correspondence
