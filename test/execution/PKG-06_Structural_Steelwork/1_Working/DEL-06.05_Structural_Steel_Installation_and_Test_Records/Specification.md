# Specification: DEL-06.05 Structural Steel Installation & Test Records

## Scope

This specification defines the requirements for **Structural Steel Installation & Test Records** within **PKG-06 Structural Steelwork**.

Provides evidence of completion, inspection, and testing for structural steel.

**Anticipated deliverable artifacts:**
- Mill certificates
- weld inspection records
- galvanizing certificates
