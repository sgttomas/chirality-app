# Specification: DEL-30.06 Performance Test Installation & Test Records

## Scope

Defines requirements for **Performance Test Installation & Test Records** for the Canola Oil Transload Facility — Phase 1.

**Purpose:** Provides evidence of completion, inspection, and testing for performance test.

**Artifacts:** Performance verification records

**Source:** Decomposition §5 PKG-30, DEL-30.06

## Requirements

### Functional Requirements

**FR-1: Throughput Capacity Verification (OBJ-2)**
- Performance test shall demonstrate facility achieves permitted capacity: 1,000,000 MT per annum
- Test duration and methodology to demonstrate annual throughput capacity — **TBD**
- **Source:** Decomposition §2 OBJ-2 (Throughput Capacity)

**FR-2: Storage Capacity Verification (OBJ-3)**
- Performance test shall verify storage capacity: 45,000 MT (3 × 15,000 MT tanks)
- Tank capacity verification by level instrumentation and calibration
- **Source:** Decomposition §2 OBJ-3 (Storage Capacity)

**FR-3: Custody Transfer Metering Accuracy (OBJ-10)**
- Performance test shall verify custody transfer metering accuracy per contractual requirements
- Meter proving, accuracy certification, flow computer verification
- **Source:** Decomposition §2 OBJ-10 (Custody Transfer Accuracy)

**FR-4: Operational Flexibility Demonstration (OBJ-4)**
- Performance test shall demonstrate tank storage mode and direct rail-to-ship mode with actual product
- **Source:** Decomposition §2 OBJ-4 (Operational Flexibility)

**FR-5: Reliability Demonstration (OBJ-1)**
- Performance test shall include continuous operation test (duration **TBD**)
- System availability and reliability verification
- **Source:** Decomposition §2 OBJ-1 (Safe & Reliable Operation)

### Performance Requirements

**PR-1: Design Capacity Achievement**
- Facility shall achieve design throughput and storage capacity
- Acceptance criteria per contract and Employer's Requirements — **TBD: location**
- **Source:** **ASSUMPTION** — Contractual performance requirements

**PR-2: Product Quality Maintenance**
- Performance test shall verify product quality maintained (food-grade CSD canola oil)
- Temperature control, contamination prevention verified
- **Source:** Decomposition §1.1 (Product: CSD canola oil); **ASSUMPTION** — Product quality requirements

**PR-3: Operational Readiness**
- Performance test completion demonstrates facility ready for handover to operations
- **Source:** **ASSUMPTION** — Performance test as operational readiness gate

### Interface Requirements

**IR-1: IST Prerequisite**
- Performance testing requires DEL-30.05 IST complete
- **Source:** DEL-30.02 Commissioning Plan; DEL-30.05

**IR-2: Operational Handover**
- Performance test records support operational handover (PKG-35 Training & Handover)
- **Source:** Decomposition §4 PKG-35

**IR-3: Contractual Acceptance**
- Performance test records provide evidence for contractual acceptance
- **Source:** **ASSUMPTION** — Performance test for acceptance

### Quality Requirements

**QR-1: Approval and Acceptance**
- Performance test records require QA/QC Manager, Commissioning Manager, Operations Manager, Employer sign-offs
- **Source:** **ASSUMPTION** — Performance test governance

**QR-2: Record Quality**
- Performance test records shall provide objective evidence of capacity achievement and acceptance criteria met
- **Source:** **ASSUMPTION**
