# Specification: DEL-18.04 Lighting Data Sheet Package

## Scope

This specification defines the requirements for the **Lighting Data Sheet Package** deliverable within **PKG-18 Lighting** for the Canola Oil Transload Facility.

**Purpose:** Defines and substantiates lighting in accordance with ER requirements. *(Source: _CONTEXT.md, Decomposition DEL-18.04 description)*

**Package Scope:** Interior and exterior LED lighting, lighting controls, emergency lighting. *(Source: Decomposition PKG-18 scope)*

**Deliverable Artifacts:**
- Light fixture data sheets
- Pole data sheets

*(Source: _CONTEXT.md, Decomposition DEL-18.04 anticipated artifacts)*

**Inclusions:**
- Manufacturer product data sheets for all LED fixture types shown on lighting drawings (DEL-18.01)
- Manufacturer product data sheets for all lighting pole types
- IES photometric data files (IESNA LM-63 format) for all fixtures
- Product certifications (CSA C22.2, UL, or equivalent)
- Installation instructions and dimensional drawings
- Warranty documentation

**Exclusions:**
- Equipment procurement (contractor responsibility using data sheets)
- Shop drawings for custom fabrications (unless required by specification DEL-18.02)
- As-built documentation (see DEL-18.05)

## Requirements

### Functional Requirements

**FR-1: Data Sheet Coverage**
- Data sheet package shall include product data for all fixture types shown on lighting drawings (DEL-18.01) — **ASSUMPTION**
- Data sheet package shall include product data for all pole types shown on lighting drawings (DEL-18.01) — **ASSUMPTION**
- Each data sheet shall clearly identify manufacturer, model number, and product description — **ASSUMPTION**

**FR-2: Performance Data**
- Fixture data sheets shall provide:
  - Photometric performance: lumen output, efficacy (lm/W), distribution type, beam angle — **ASSUMPTION**
  - IES photometric file (LM-63 format) for use in calculations (DEL-18.03) — **ASSUMPTION**
  - Electrical characteristics: voltage, wattage, current, power factor, harmonic distortion — **ASSUMPTION**
  - LED specifications: CCT, CRI, lumen maintenance (L70 hours per IES LM-80/TM-21) — **TBD**
  - Environmental ratings: IP rating, operating temperature range, corrosion resistance — **ASSUMPTION**

**FR-3: Compliance Documentation**
- Data sheets shall include or reference product certifications demonstrating compliance with:
  - CSA C22.2 No. 250.0 or equivalent for LED equipment — **TBD**
  - UL 1598 or CSA equivalent for luminaires — **TBD**
  - Hazardous location certifications (Class I Div/Zone ratings) for fixtures in classified areas — **TBD**

**FR-4: Installation and Mounting Data**
- Data sheets shall provide dimensional drawings showing:
  - Overall dimensions and weight
  - Mounting details and hardware requirements
  - Electrical connection locations and requirements
  - Clearance requirements for heat dissipation and maintenance access — **ASSUMPTION**

### Performance Requirements

**PR-1: Specification Compliance**
- All equipment data sheets shall demonstrate compliance with lighting technical specification (DEL-18.02) requirements — **ASSUMPTION**
- Performance values (lumens, efficacy, CRI, CCT, etc.) shall meet or exceed specification minimums — **ASSUMPTION**
- Material and finish requirements per specification shall be confirmed in data sheets — **ASSUMPTION**

**PR-2: Calculation Basis**
- IES photometric files shall be suitable for use in lighting design calculations (DEL-18.03) — **ASSUMPTION**
- Photometric data shall be from IES LM-79 laboratory testing (independent test lab preferred) — **ASSUMPTION**
- Fixture lumen output in data sheets shall match values used in calculations — **ASSUMPTION**

**PR-3: Environmental Suitability**
- Fixtures shall be rated for marine/industrial environment:
  - Operating temperature range -20°C to +40°C minimum — **TBD** — **ASSUMPTION**
  - IP65+ for exterior fixtures, IP54+ for interior industrial areas — **TBD** — **ASSUMPTION**
  - Corrosion-resistant materials and finishes for outdoor exposure — **ASSUMPTION**

**PR-4: Warranty Requirements**
- Warranty documentation shall demonstrate:
  - Minimum 5-year warranty for fixtures — **TBD** — **ASSUMPTION**: Per specification DEL-18.02
  - Minimum 3-year warranty for LED drivers — **TBD** — **ASSUMPTION**
  - Warranty coverage (materials, workmanship, lumen maintenance) — **TBD**

### Interface Requirements

**IR-1: Lighting Design Drawings (DEL-18.01)**
- Data sheets shall be provided for all fixture and pole types shown on drawings — **ASSUMPTION**
- Equipment tags and model numbers shall match fixture schedule on drawings — **ASSUMPTION**

**IR-2: Lighting Technical Specification (DEL-18.02)**
- Data sheets shall demonstrate compliance with all specification requirements — **ASSUMPTION**
- If proposed equipment differs from specified, "or equal" submittals shall demonstrate equivalent performance — **TBD**

**IR-3: Lighting Design Calculations (DEL-18.03)**
- IES photometric files from data sheets shall be compatible with calculation software — **ASSUMPTION**
- Fixture lumen output and distribution shall match calculation assumptions — **ASSUMPTION**

**IR-4: Procurement and Construction**
- Data sheets shall provide sufficient information for accurate cost estimating and procurement — **ASSUMPTION**
- Installation data shall enable proper installation planning and execution — **ASSUMPTION**
