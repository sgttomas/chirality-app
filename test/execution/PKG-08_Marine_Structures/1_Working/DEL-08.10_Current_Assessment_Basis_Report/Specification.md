# Specification: DEL-08.10 Current Assessment Basis Report

## Scope

Defines requirements for **Current Assessment Basis Report**: minimum content, data quality, methodology documentation, and traceability.

**Deliverable intent:** Documents analysis and results for current assessment basis report required for design verification and approvals. *(Source: Decomposition line 290)*

### Inclusions
- Assessment methodology for design current determination
- Input data sources and quality assessment
- Design current values for engineering use

### Exclusions
- Mooring analysis (DEL-08.09)
- Structural load calculations (DEL-08.03)
- Debris loading analysis (DEL-08.11)

---

## Requirements

### R-001 — Minimum Report Content

*(Source: Decomposition line 290)*

| Component | Required Content |
|---|---|
| Current assessment methodology | Approach for deriving design currents |
| Inputs | Data sources, measurements, models, assumptions |
| Results summary | Design current values and applicability |

**Rationale:** `Guidance.md` § Requirement Rationale Map, R-001

---

### R-002 — Current Data Requirements

**Document design current parameters:**

| Parameter | Requirement |
|---|---|
| Surface current velocity | Design value (m/s) with statistical basis |
| Current direction | Predominant directions and design cases |
| Tidal variation | Flood and ebb current values |
| Seasonal variation | Freshet and normal conditions |
| Depth profile | If significant variation exists |
| Return period | Design event return period basis |

**Rationale:** `Guidance.md` § Requirement Rationale Map, R-002

---

### R-003 — Data Source and Quality

**Document:**

| Element | Requirement |
|---|---|
| Data source ID | Measurements, models, literature |
| Data period | Duration of records/simulation |
| Data quality | Reliability and limitations assessment |
| Source references | Full citations |

**Rationale:** `Guidance.md` § Requirement Rationale Map, R-003

### R-004 — Methodology Documentation

**Document:**

| Element | Requirement |
|---|---|
| Assessment approach | How design values were derived |
| Statistical methods | Extreme value, percentiles, or other |
| Assumptions | Explicitly listed with rationale |
| Limitations | Known limitations |
| Applicability | How results should be applied |
