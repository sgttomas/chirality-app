# Specification: DEL-02.02 Earthworks Technical Specification

## Scope

This deliverable defines the requirements for the Earthworks Technical Specification within PKG-02 Earthworks & Ground Improvement. Source: _CONTEXT.md; decomposition DEL-02.02 entry (location TBD).

**Included:**
- Performance, materials, and workmanship requirements for earthworks
- Ground improvement requirements (methods, materials, acceptance criteria)
- Geotextile requirements (properties, installation, testing)
- Quality control and testing requirements
- All specification content necessary to support field execution per PKG-02 scope

Package scope includes Site grading, excavation, filling, ground improvements, geotechnical works, surveys. Source: decomposition PKG-02 scope (location TBD).

**Excluded:**
