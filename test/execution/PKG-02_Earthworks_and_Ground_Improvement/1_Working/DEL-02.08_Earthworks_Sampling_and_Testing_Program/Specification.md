# Specification: DEL-02.08 Earthworks Sampling & Testing Program

## Scope

This deliverable defines the requirements for the Earthworks Sampling & Testing Program within PKG-02 Earthworks & Ground Improvement. Source: _CONTEXT.md; decomposition DEL-02.08 entry (location TBD).

**Included:**
- Sampling plan (objectives, locations, procedures, chain of custody, sample types)
- Frequency matrix (test types, minimum frequencies, phase-specific frequencies, triggers)
- Laboratory testing scope (classification, moisture-density, strength, chemical, qualifications, standards)
- Field testing scope (density, moisture, proof rolling, visual inspections, qualifications, calibration)

**Package Context:**
Package scope includes Site grading, excavation, filling, ground improvements, geotechnical works, surveys. Source: decomposition PKG-02 scope (location TBD).

**Excluded:**
This decomposition covers the Contractor's scope of work only. Employer-responsible items (permits obtained by Employer, nitrogen skid supply) are excluded except for interface connections. Source: decomposition Section 1.2 Scope Focus (location TBD).
