# Specification: DEL-02.04 Geotechnical Reports

## Scope

This deliverable defines the requirements for Geotechnical Reports within PKG-02 Earthworks & Ground Improvement. Source: _CONTEXT.md; decomposition DEL-02.04 entry (location TBD).

**Included:**
- Geotechnical investigation report documenting subsurface conditions and design recommendations
- Ground improvement design report (if required)
- Engineering analysis and design parameters for earthworks and foundations
- Professional engineering documentation with appropriate seals/stamps

**Package Context:**
Package scope includes Site grading, excavation, filling, ground improvements, geotechnical works, surveys. Source: decomposition PKG-02 scope (location TBD).

**Excluded:**
This decomposition covers the Contractor's scope of work only. Employer-responsible items excluded except for interface connections. Source: decomposition Section 1.2 Scope Focus (location TBD).
