# Specification: DEL-02.09 In-Situ Compaction Verification Plan & Results

## Scope

This deliverable defines the requirements for the In-Situ Compaction Verification Plan & Results within PKG-02 Earthworks & Ground Improvement. Source: _CONTEXT.md; decomposition DEL-02.09 entry (location TBD).

**Included:**
- Compaction verification plan (objectives, acceptance criteria, reference values, methods, frequency, trial compaction, equipment, nonconformance procedure, approvals)
- Test results (field density records, trial compaction, test location plan, statistical summary, proof rolling)
- Nonconformance reports and closeout (failed tests, root cause, corrective action, retest, closure)

**Package Context:**
Package scope includes Site grading, excavation, filling, ground improvements, geotechnical works, surveys. Source: decomposition PKG-02 scope (location TBD).

**Excluded:**
This decomposition covers the Contractor's scope of work only. Employer-responsible items (permits obtained by Employer, nitrogen skid supply) are excluded except for interface connections. Source: decomposition Section 1.2 Scope Focus (location TBD).
